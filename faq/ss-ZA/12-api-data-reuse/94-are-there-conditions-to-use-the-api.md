Yonkhe imibhalo lemayelana nekusetjentiswa kwe-API ingatfolakala ekhasini lemibhalo ye-API, kepha nasi sifinyeto lesisheshako:

- I-database yeMaciniso Ekudla Lavulekile iyatfolakala njengedatha levulekile ngaphasi kweLayisensi Yedatha Levulekile (i-ODbL), bona https://world.openfoodfacts.org/imigomo-yekusebentisa kute utfole imininingwane lesemtsetfweni. Letimo letimbili tiyi-attribution kanye nekwabelana-lokufanako. Uma uhlanganisa idatha levela ku-Open Food Facts nalamanye ema-database, khona-ke i-ODbL idzinga kutsi le-database lephumako kumele ikhululwe njengedatha levulekile futsi. Kuphindze kusho kutsi ungahlanganisa idatha kuphela nemithombo lebeyitawuvumela kusabalalisa kabusha lokunjalo.

- Kufanele uhlale usebentisa i-User-Agent lekhetsekile uma wenta lucingo lwe-API kute ubone i-app yakho.

- Imikhawulo yesilinganiso iyaphocelelwa ku-API ngayinye yekugcina.
