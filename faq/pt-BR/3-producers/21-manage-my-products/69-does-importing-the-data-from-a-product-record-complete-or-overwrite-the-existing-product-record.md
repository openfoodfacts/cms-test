Usamos o código de barras para identificar os produtos. Se você importar um registro com o mesmo código de barras de um registro existente, as informações serão mescladas.

Para dados como lista de ingredientes e valores nutricionais (apenas um valor correto é possível), os dados enviados pela plataforma do produtor substituirão os dados existentes.

Para dados como rótulos/categorias/marcas (vários valores corretos possíveis), as informações são mescladas. Caso os dados estejam incorretos na plataforma pública, você pode modificar a ficha do produto na plataforma pública para remover os valores incorretos.
