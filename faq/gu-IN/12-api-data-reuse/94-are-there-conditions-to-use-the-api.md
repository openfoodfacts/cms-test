API ઉપયોગ વિશેના બધા દસ્તાવેજો API દસ્તાવેજીકરણ પૃષ્ઠ પર મળી શકે છે, પરંતુ અહીં એક ટૂંકી સારાંશ છે:

- ઓપન ફૂડ ફેક્ટ્સ ડેટાબેઝ ઓપન ડેટાબેઝ લાયસન્સ (ODbL) હેઠળ ઓપન ડેટા તરીકે ઉપલબ્ધ છે, કાનૂની વિગતો માટે https://world.openfoodfacts.org/terms-of-use જુઓ. બે શરતો એટ્રિબ્યુશન અને શેર-એલાઈક છે. જો તમે ઓપન ફૂડ ફેક્ટ્સમાંથી ડેટાને અન્ય ડેટાબેઝ સાથે જોડો છો, તો ODbL માટે જરૂરી છે કે પરિણામી ડેટાબેઝને ઓપન ડેટા તરીકે પણ રિલીઝ કરવામાં આવે. તેનો અર્થ એ પણ છે કે તમે ડેટાને ફક્ત એવા સ્ત્રોતો સાથે જ જોડી શકો છો જે આવા પુનઃવિતરણને મંજૂરી આપે.

- તમારી એપને ઓળખવા માટે API કોલ્સ કરતી વખતે તમારે હંમેશા કસ્ટમ યુઝર-એજન્ટનો ઉપયોગ કરવો જોઈએ.

- દરેક API એન્ડપોઇન્ટ માટે રેટ-લિમિટ લાગુ કરવામાં આવે છે.
