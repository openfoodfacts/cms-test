Khombo ra kona a swi si koteka ku lavisisa hi ku olova eka vito ra xiendliwa ntsena naswona hi ku kongoma hi ku tirhisa API.

Ku tirhisa xisefo eka xiyenge swi nga ku pfuna ku endla leswaku ku lavisisa ka wena ku kongoma swinene hambiswiritano.
