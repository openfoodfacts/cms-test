Matsalwa hinkwawo mayelana na matirhiselo ya API ya nga kumeka eka tluka ra matsalwa ya API, kambe hi lowu nkatsakanyo wo hatlisa:

- Dathabeyisi ya Tinhla ta Swakudya leti Pfulekeke yi kumeka tanihi datha leyi pfulekeke ehansi ka Layisense ya Dathabeyisi leyi Pfulekeke (ODbL), vona https://world.openfoodfacts.org/terms-of-use ku kuma vuxokoxoko bya nawu. Swiyimo leswimbirhi i attribution na share-alike. Loko u hlanganisa data kusuka eka Open Food Facts na ti database tin’wana, kutani ODbL yi lava leswaku database leyi humelelaka yifanele ku humesiwa tani hi data leyi pfulekeke na yona. Swi tlhela swi vula leswaku u nga hlanganisa datha ntsena na swihlovo leswi nga pfumelelaka ku hangalasiwa nakambe ko tano.

- U fanele ku tshama u tirhisa Mutirhisi-Muyimeri wa ntolovelo loko u endla swikombelo swa API ku kuma app ya wena.

- Swipimelo swa mpimo swi sindzisiwa eka ndhawu yin’wana na yin’wana yo hetelela ya API.
