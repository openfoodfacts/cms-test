Hal itu mungkin bergantung pada negara, apakah jaringan internet lambat atau mahal. Ukuran atau lebar apa pun yang melebihi 5000 piksel mungkin tidak terlalu berguna. dan jika Anda dapat mendeteksi bahwa jaringan lambat, maka gambar 2000 piksel pun akan sangat membantu (tentu lebih baik daripada tidak memiliki gambar sama sekali!).
