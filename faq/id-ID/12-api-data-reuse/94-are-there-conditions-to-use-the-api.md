Semua dokumentasi tentang penggunaan API dapat ditemukan di halaman dokumentasi API, tetapi berikut ringkasan singkatnya:

- Basis data Open Food Facts tersedia sebagai data terbuka di bawah Lisensi Basis Data Terbuka (ODbL), lihat https://world.openfoodfacts.org/terms-of-use untuk detail hukumnya. Kedua kondisi tersebut adalah atribusi dan berbagi secara setara. Jika Anda menggabungkan data dari Open Food Facts dengan basis data lain, maka ODbL mensyaratkan bahwa basis data yang dihasilkan juga harus dirilis sebagai data terbuka. Ini juga berarti bahwa Anda hanya dapat menggabungkan data dengan sumber yang mengizinkan pendistribusian ulang tersebut.

- Anda harus selalu menggunakan User-Agent khusus saat melakukan panggilan API untuk mengidentifikasi aplikasi Anda.

- Pembatasan laju permintaan diberlakukan untuk setiap titik akhir API.
