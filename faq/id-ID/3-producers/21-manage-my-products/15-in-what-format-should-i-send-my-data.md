Templat tersebut disediakan oleh Open Food Facts.

Namun, format spreadsheet apa pun (xlsx, csv) dapat diimpor ke dalam platform ini.Algoritma kami dirancang untuk secara otomatis mencocokkan atribut dari file Anda dengan atribut yang ada di Open Food Facts. Berhati-hatilah untuk selalu memeriksa kembali tahap pencocokan karena Anda mungkin perlu menyesuaikan beberapa di antaranya secara manual.

Jika informasi Anda terpecah-pecah di antara beberapa spreadsheet, Anda dapat mengimpornya secara berurutan - pastikan untuk melakukan pencocokan jika formatnya berubah.

Anda juga dapat mengimpor file Excel selain impor dari EQUADIS atau AGENA3000, jika data tersebut belum ada dalam sistem informasi Anda. Dalam kasus ini, jangan lupa untuk meminta ekspor ke basis data publik dari antarmuka.
