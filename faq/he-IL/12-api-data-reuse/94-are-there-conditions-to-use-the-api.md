את כל התיעוד אודות השימוש ב-API ניתן למצוא בדף תיעוד ה-API, אך הנה סיכום קצר:

- מסד הנתונים Open Food Facts זמין כנתונים פתוחים תחת רישיון מסד נתונים פתוח (ODbL). ראה https://world.openfoodfacts.org/terms-of-use לפרטים משפטיים. שני התנאים הם ייחוס ושיתוף זהה. אם משלבים נתונים מ-Open Food Facts עם מאגרי מידע אחרים, ODbL דורש שגם מסד הנתונים המתקבל יפורסם כנתונים פתוחים. זה גם אומר שניתן לשלב את הנתונים רק עם מקורות שיאפשרו חלוקה מחדש כזו.

- עליך תמיד להשתמש בסוכן משתמש מותאם אישית בעת ביצוע קריאות API כדי לזהות את האפליקציה שלך.

- מגבלות קצב נאכפות עבור כל נקודת קצה של API.
