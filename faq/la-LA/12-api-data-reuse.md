## Suntne uspiam in documentis commendationes de magnitudine idonea imaginibus immissis?

Id fortasse a terris pendet, utrum rete lentum an carum sit. Quidquid supra 5000 pixellorum pondere aut altitudine est probabiliter non admodum utile est. Et si aliquo modo deprehendere potes retem tardum esse, tum etiam imago duorum milium pixellorum optima esset (certe melior quam imaginem non habere!).

## Quid de cibo sine codicibus linearibus?

"Open Food Facts" solum informationem de cibo involucris continet. Pro valoribus mediis fructuum et olerum (exempli gratia, lycopersicorum vel bananarum) aliorumque ciborum, unam ex officialibus nationalibus databasibus nutritionis uti potes.

**Nota:** Index infra nonnullas ex maximis momenti nationalibus notitiarum cibariorum collectionibus continet. Si aliam aliquam basim datorum in indice includi debere putas, nobis scribe ad: [[https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## Possumne nomen producti accuratum per API quaerere?

Infeliciter nondum possibile est facile per nomen producti solum et accurate per API quaerere.

Usus tamen filtro per categoriam fortasse adiuvabit te ut investigationem tuam accuratiorem reddas.

## Quomodo ad notitias pro meis inceptis accedere/colligere possum?

In pagina principali Open Food Facts, in angulo sinistro superiore scrinii, est index volubilis. In fundo eius, optionem "quaestionis provectae" invenies, quam premere potes. Deinde tibi incumbit determinare quae criteria ad proiecta tua maxime pertinentia sint. Postquam electum est, eventus obtentos depromere poteris deorsum in fundo paginae volvendo et "Depromere eventus" premendo.

Consulere etiam potes:

- Documentatio nostra API:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Conditiones Usus "Open Food Facts":[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- De nostris datis:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## Suntne condiciones ad API utendum?

Omnis documentatio de usu API in [pagina documentationis API](https://openfoodfacts.github.io/openfoodfacts-server/api/) inveniri potest, sed hic brevis summarium est:

- Index notitiarum de cibis apertis (Open Food Facts) sub Licentia Datarum Apertarum (ODbL) ut notitiae apertae praesto est; vide [[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) pro singulis legalibus. Duae condiciones sunt attributio et communicatio-aequalis. Si notitias ex "Open Food Facts" cum aliis databasibus coniungis, tunc ODbL postulat ut databasis inde orta etiam ut data aperta divulgetur. Significat etiam te posse data coniungere solum cum fontibus qui talem redistributionem permitterent.

- Semper debes agentem usoris proprium adhibere cum invocationes API facis ad applicationem tuam cognoscendam.

- Limites celeritatis pro singulis terminis API imponuntur.

## Quomodo ad notitias historicas accedere possum?

In praesenti, collectionem datorum historicorum (JSONL, MongoDB, CSV) non offerimus.

Attamen, pro singulis productis, versiones priores datorum producti per API vel in pagina producti per revisiones accedere licet.

Quotiescumque productum renovatur, nova recensio (digito crescente ab 1 incipiens) creatur.

Exempli gratia, ad primam revisionem (=primam versionem producti) huius producti obtinendam, utere

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Similiter, parametrus `rev` cum API adhiberi potest:

[[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)]
