API Open Food Facts 98% eadem est ac API Open Beauty Facts, ergo si documentatione API Open Food Facts uteris, visionem satis claram API Open Beauty Facts habere debes.

Nota bene functiones sicut Tabulae Scientiae, Eco-Score, Nutri-Score non sustineri (vel propter argumentum vel propter vetustam basim codicis).
