API-ის გამოყენების შესახებ ყველა დოკუმენტაცია შეგიძლიათ იხილოთ API დოკუმენტაციის გვერდზე, თუმცა აქ მოცემულია მოკლე შეჯამება:

- ღია საკვების ფაქტების მონაცემთა ბაზა ხელმისაწვდომია ღია მონაცემების სახით ღია მონაცემთა ბაზის ლიცენზიით (ODbL), იურიდიული დეტალებისთვის იხილეთ https://world.openfoodfacts.org/terms-of-use. ორი პირობაა ატრიბუცია და თანაბარი გაზიარება. თუ Open Food Facts-ის მონაცემებს სხვა მონაცემთა ბაზებთან გააერთიანებთ, მაშინ ODbL მოითხოვს, რომ შედეგად მიღებული მონაცემთა ბაზაც ღია მონაცემების სახით გამოქვეყნდეს. ეს ასევე ნიშნავს, რომ მონაცემების გაერთიანება მხოლოდ იმ წყაროებთან შეგიძლიათ, რომლებიც ასეთი გადანაწილების საშუალებას მოგცემთ.

- თქვენი აპლიკაციის იდენტიფიცირებისთვის API ზარების შესრულებისას ყოველთვის უნდა გამოიყენოთ მორგებული მომხმარებლის აგენტი.

- სიჩქარის ლიმიტები დაწესებულია თითოეული API საბოლოო წერტილისთვის.
