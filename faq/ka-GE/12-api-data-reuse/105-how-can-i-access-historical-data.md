ამჟამად, ჩვენ არ ვთავაზობთ ისტორიული მონაცემების დემპს (JSONL, MongoDB, CSV).

თუმცა, ცალკეული პროდუქტებისთვის, პროდუქტის მონაცემების წინა ვერსიებზე წვდომა შესაძლებელია API-ის გამოყენებით ან პროდუქტის გვერდზე რედაქტირებების გამოყენებით.

ყოველ ჯერზე, როდესაც პროდუქტი განახლდება, იქმნება ახალი ვერსია (ზრდადი ციფრი 1-დან).
