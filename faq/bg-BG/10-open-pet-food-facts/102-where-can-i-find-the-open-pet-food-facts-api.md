API на Open Pet Food Facts е 98% същият като API на Open Food Facts, така че ако използвате документацията на API на Open Food Facts, би трябвало да имате доста ясна представа за API на Open Pet Food Facts.

Обърнете внимание, че функции като Knowledge Panels, Eco-Score, Nutri-Score не се поддържат (или поради темата, или поради старата кодова база).
