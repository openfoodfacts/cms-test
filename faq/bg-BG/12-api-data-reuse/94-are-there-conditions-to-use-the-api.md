Цялата документация относно използването на API може да бъде намерена на страницата с документация за API, но ето едно кратко резюме:

- Базата данни „Отворени факти за храните“ е достъпна като отворени данни съгласно лиценза за отворена база данни (ODbL), вижте https://world.openfoodfacts.org/terms-of-use за правни подробности. Двете условия са приписване и споделяне по споделени условия. Ако комбинирате данни от Open Food Facts с други бази данни, тогава ODbL изисква получената база данни също да бъде публикувана като отворени данни. Това също означава, че можете да комбинирате данните само с източници, които биха позволили такова преразпределение.

- Винаги трябва да използвате персонализиран потребителски агент, когато извършвате API повиквания, за да идентифицирате приложението си.

- Ограниченията на скоростта се прилагат за всяка крайна точка на API.
