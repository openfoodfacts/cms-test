## Adakah terdapat cadangan di mana-mana sahaja dalam dokumentasi tentang saiz yang sesuai untuk foto yang dimuat naik?

Itu mungkin bergantung pada negara, sama ada rangkaian perlahan atau mahal. Apa-apa sahaja yang melebihi 5000 piksel dari segi berat atau tinggi mungkin tidak begitu berguna. dan jika anda entah bagaimana dapat mengesan bahawa rangkaian itu perlahan, maka imej 2000 piksel pun sudah memadai (sudah tentu lebih baik daripada tidak mempunyai imej!)

## Bagaimana pula dengan makanan tanpa kod bar?

Fakta Makanan Terbuka hanya mengandungi maklumat tentang makanan yang dibungkus. Untuk nilai purata hasil (contohnya, tomato atau pisang) dan produk makanan lain, anda boleh menggunakan salah satu pangkalan data pemakanan kebangsaan rasmi.

**Nota:** Senarai di bawah mengandungi beberapa pangkalan data makanan kebangsaan yang paling penting. Jika anda berpendapat bahawa beberapa pangkalan data lain perlu dimasukkan ke dalam senarai ini, sila hubungi kami di: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## Bolehkah saya mencari nama produk yang tepat dengan API?

Malangnya, pencarian pada nama produk sahaja dan tepat melalui API masih belum dapat dilakukan dengan mudah.

Walau bagaimanapun, penggunaan penapis pada kategori mungkin membantu anda menjadikan carian anda lebih tepat.

## Bagaimanakah saya boleh mengakses/mengumpul data untuk projek saya?

Pada halaman utama Open Food Facts, di penjuru kiri sebelah atas skrin, terdapat menu skrol. Di bahagian bawahnya, anda akan menemui pilihan "carian lanjutan", yang boleh anda klik. Kemudian terpulang kepada anda untuk menentukan kriteria mana yang paling relevan dengan projek anda. Sebaik sahaja dipilih, anda boleh memuat turun keputusan yang diperoleh dengan menatal ke bawah di bahagian bawah halaman dan mengklik pada "Muat turun keputusan".

Anda juga boleh berunding:

- Dokumentasi API kami:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Syarat Penggunaan Open Food Facts:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Pada data kami:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## Adakah terdapat syarat untuk menggunakan API?

Semua dokumentasi tentang penggunaan API boleh didapati di [halaman dokumentasi API](https://openfoodfacts.github.io/openfoodfacts-server/api/), tetapi berikut ialah ringkasan ringkas:

- Pangkalan data Fakta Makanan Terbuka tersedia sebagai data terbuka di bawah Lesen Pangkalan Data Terbuka (ODbL), lihat [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) untuk butiran undang-undang. Kedua-dua syarat tersebut adalah atribusi dan perkongsian sama. Jika anda menggabungkan data daripada Fakta Makanan Terbuka dengan pangkalan data lain, maka ODbL menghendaki pangkalan data yang terhasil dikeluarkan sebagai data terbuka juga. Ini juga bermakna anda hanya boleh menggabungkan data dengan sumber yang membenarkan pengagihan semula sedemikian.

- Anda mesti **sentiasa** menggunakan Ejen Pengguna tersuai semasa melakukan panggilan API untuk mengenal pasti aplikasi anda.

- Had kadar dikuatkuasakan untuk setiap titik akhir API.

## Bagaimanakah saya boleh mengakses data sejarah?

Pada masa ini, kami tidak menawarkan lambakan data sejarah (JSONL, MongoDB, CSV).

Walau bagaimanapun, untuk produk individu, versi data produk sebelumnya boleh diakses menggunakan API atau pada halaman produk menggunakan semakan semula.

Setiap kali produk dikemas kini, semakan baharu (digit yang meningkat bermula dari 1) akan dicipta.

Contohnya, untuk mendapatkan semakan pertama (=versi produk pertama) produk ini, gunakan

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Begitu juga, parameter rev boleh digunakan dengan API:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
