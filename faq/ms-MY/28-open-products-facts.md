## Bagaimana untuk mengakses Fakta Produk Terbuka pada iPhone?

Fakta Produk Terbuka tersedia dalam aplikasi Fakta Makanan Terbuka utama. Integrasi ini membolehkan pengguna iPhone mengakses kedua-dua pangkalan data melalui satu aplikasi yang mudah.

---

## Saya ingin menyokong Fakta Produk Terbuka. Apa yang boleh saya lakukan?

Sila baca halaman wiki kami, kami telah menyenaraikan cara untuk terlibat: [https://wiki.openfoodfacts.org/Open_Products_Facts](https://wiki.openfoodfacts.org/Open_Products_Facts)

Untuk menyokong Fakta Produk Terbuka, anda boleh mempertimbangkan pilihan berikut, setiap satunya disesuaikan dengan pelbagai jenis sumbangan:

-

**Penaja Filantropi/Negeri untuk Fakta Produk Terbuka**:
Sumbangan dermawan adalah penting dalam merealisasikan visi platform data kitaran yang komprehensif, pemarkahan produk dan pengagregatan penyelesaian kitaran, merentasi lebih 20000+ kategori, matlamat utama Fakta Produk Terbuka. Banyak komponen penting dalam visi ini masih belum dibiayai, dan sokongan kewangan anda boleh memberi impak yang besar. Sumbangan anda dapat membantu membiayai pembangunan dan infrastruktur kritikal. Untuk menyumbang dari segi kewangan, sila hubungi di contact@openfoodfacts.org

-

**Penyumbang Data Menggunakan Aplikasi Fakta Produk Terbuka**:
Sumbangan data adalah penting untuk meningkatkan Fakta Produk Terbuka. Dengan menggunakan aplikasi Android untuk menambah maklumat produk baharu, mengemas kini entri sedia ada atau mengesahkan data, anda membantu mengembangkan dan memperhalusi pangkalan data. Usaha kolektif ini menyumbang kepada pembinaan sumber yang mantap dan andal untuk semua orang.

-

**Penyumbang Teknikal kepada Fakta Produk Terbuka**:
Jika anda mempunyai kemahiran teknikal dalam bidang seperti pengaturcaraan, analisis data atau reka bentuk pengalaman pengguna, kepakaran anda boleh memberi manfaat yang ketara kepada Fakta Produk Terbuka. Melibatkan diri dalam pembangunan aplikasi, menambah baik fungsinya atau mempertingkatkan antara muka pengguna merupakan sumbangan yang berharga. Untuk terlibat, sertai saluran #openproductsfacts Slack, tempat komuniti Fakta Produk Terbuka bekerjasama. Anda boleh mendapatkan pautan ke ruang kerja Slack ini di laman web Open Food Facts.

Setiap jenis sumbangan adalah penting untuk pertumbuhan dan kejayaan Open Products Facts, membantu mewujudkan platform yang lebih mampan dan bermaklumat untuk semua orang.

## Di manakah saya boleh menemui API Fakta Produk Terbuka?

API Fakta Produk Terbuka adalah 98% sama dengan API Fakta Makanan Terbuka, jadi jika anda menggunakan dokumentasi API Fakta Makanan Terbuka, anda sepatutnya mempunyai pandangan yang agak jelas tentang API Fakta Produk Terbuka. Kami juga telah menulis tutorial khusus untuk membolehkan pelaksanaan imbasan universal dalam aplikasi anda.

[https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/](https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/)
