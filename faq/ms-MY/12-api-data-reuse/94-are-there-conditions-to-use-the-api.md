Semua dokumentasi tentang penggunaan API boleh didapati di halaman dokumentasi API, tetapi berikut adalah ringkasan ringkas:

- Pangkalan data Fakta Makanan Terbuka tersedia sebagai data terbuka di bawah Lesen Pangkalan Data Terbuka (ODbL), lihat https://world.openfoodfacts.org/terms-of-use untuk butiran undang-undang. Kedua-dua syarat tersebut adalah atribusi dan perkongsian sama. Jika anda menggabungkan data daripada Fakta Makanan Terbuka dengan pangkalan data lain, maka ODbL menghendaki pangkalan data yang terhasil dikeluarkan sebagai data terbuka juga. Ini juga bermakna anda hanya boleh menggabungkan data dengan sumber yang membenarkan pengagihan semula sedemikian.

- Anda mesti sentiasa menggunakan Ejen Pengguna tersuai semasa melakukan panggilan API untuk mengenal pasti aplikasi anda.

- Had kadar dikuatkuasakan untuk setiap titik akhir API.
