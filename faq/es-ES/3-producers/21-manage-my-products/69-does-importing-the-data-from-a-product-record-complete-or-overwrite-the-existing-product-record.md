Utilizamos el código de barras para identificar productos, si importa un registro con el mismo código de barras que un registro existente, la información se fusionará.

Para datos como la lista de ingredientes o los valores nutricionales (solo es posible un valor correcto), los datos enviados a través de la plataforma del productor sobrescribirán los datos existentes.

Para datos como etiquetas/categorías/marcas (varios valores correctos posibles), la información se fusiona. Si los datos son incorrectos en la plataforma pública, puede modificar la hoja de producto en la plataforma pública para eliminar los valores incorrectos.
