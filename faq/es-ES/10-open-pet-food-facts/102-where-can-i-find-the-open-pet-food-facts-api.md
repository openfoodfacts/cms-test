La API de Open Pet Food Facts es 98% igual que la API de Open Food Facts, por lo que si utiliza la documentación de la API de Open Food Facts, debería tener una visión bastante clara de la API de Open Pet Food Facts.

Tenga en cuenta que funciones como los paneles de conocimiento, Eco-Score y Nutri-Score no son compatibles (ya sea debido al tema o al código base antiguo).

https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/
