## උඩුගත කරන ලද ඡායාරූප සඳහා හොඳ ප්‍රමාණය කුමක්ද යන්න පිළිබඳ නිර්දේශ ලේඛනයේ කොතැනක හෝ තිබේද?

ජාලය මන්දගාමී හෝ මිල අධික නම්, එය රටවල් මත රඳා පවතී. බර හෝ උස පික්සල 5000 ට වැඩි ඕනෑම දෙයක් එතරම් ප්‍රයෝජනවත් නොවනු ඇත. ඔබට කෙසේ හෝ ජාලය මන්දගාමී බව හඳුනාගත හැකි නම්, පික්සල 2000 ක රූපයක් පවා විශිෂ්ට වනු ඇත (නිසැකවම රූපයක් නොතිබීමට වඩා හොඳයි!)

## බාර්කෝඩ් නැති ආහාර ගැන කුමක් කිව හැකිද?

Open Food Facts හි අඩංගු වන්නේ ඇසුරුම් කළ ආහාර පිළිබඳ තොරතුරු පමණි. නිෂ්පාදන (උදාහරණයක් ලෙස තක්කාලි හෝ කෙසෙල්) සහ අනෙකුත් ආහාර නිෂ්පාදනවල සාමාන්‍ය අගයන් සඳහා, ඔබට නිල ජාතික පෝෂණ දත්ත සමුදායන්ගෙන් එකක් භාවිතා කළ හැකිය.

## මට API භාවිතයෙන් නිවැරදි නිෂ්පාදන නාමයක් සෙවිය හැකිද?

අවාසනාවකට, නිෂ්පාදන නාමයෙන් පමණක් සහ හරියටම API හරහා සෙවීමට තවමත් නොහැකි වී තිබේ.

කෙසේ වෙතත්, කාණ්ඩය මත පෙරහනක් භාවිතා කිරීමෙන් ඔබේ සෙවීම වඩාත් නිවැරදි කිරීමට උපකාරී වේ.

## මගේ ව්‍යාපෘති සඳහා දත්ත ප්‍රවේශ කර ගන්නේ/එකතු කරන්නේ කෙසේද?

Open Food Facts හි ප්‍රධාන පිටුවේ, තිරයේ ඉහළ වම් කෙළවරේ, අනුචලන මෙනුවක් ඇත. එහි පහළින්, ඔබට "උසස් සෙවුම්" විකල්පය සොයාගත හැකි අතර, එය මත ක්ලික් කළ හැකිය. ඉන්පසු ඔබේ ව්‍යාපෘතියට/ව්‍යාපෘතිවලට වඩාත්ම අදාළ වන්නේ කුමන නිර්ණායකද යන්න තීරණය කිරීම ඔබට භාරයි. තෝරාගත් පසු, පිටුවේ පහළින් පහළට අනුචලනය කර "ප්‍රතිඵල බාගන්න" මත ක්ලික් කිරීමෙන් ඔබට ලබාගත් ප්‍රතිඵල බාගත කිරීමට හැකි වනු ඇත.

ඔබට ද උපදෙස් ලබා ගත හැකිය:

- අපගේ API ලේඛන:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- විවෘත ආහාර කරුණු වල භාවිත නියමයන්:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- අපගේ දත්ත මත:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## API භාවිතා කිරීමට කොන්දේසි තිබේද?

API භාවිතය පිළිබඳ සියලුම ලියකියවිලි [API ලේඛන පිටුවෙන්](https://openfoodfacts.github.io/openfoodfacts-server/api/සොයා ගත හැක), නමුත් මෙන්න ඉක්මන් සාරාංශයක්:

- විවෘත ආහාර කරුණු දත්ත සමුදාය විවෘත දත්ත සමුදා බලපත්‍රය (ODbL) යටතේ විවෘත දත්ත ලෙස ලබා ගත හැකිය, නීතිමය විස්තර සඳහා [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) බලන්න. කොන්දේසි දෙක ආරෝපණය සහ බෙදා ගැනීම සමාන වේ. ඔබ විවෘත ආහාර කරුණු වලින් ලැබෙන දත්ත අනෙකුත් දත්ත සමුදායන් සමඟ ඒකාබද්ධ කරන්නේ නම්, ODbL විසින් ප්‍රතිඵලයක් ලෙස ලැබෙන දත්ත සමුදාය විවෘත දත්ත ලෙස නිකුත් කළ යුතු බව අවශ්‍ය කරයි. තවද එයින් අදහස් වන්නේ ඔබට දත්ත ඒකාබද්ධ කළ හැක්කේ එවැනි නැවත බෙදා හැරීමට ඉඩ සලසන මූලාශ්‍ර සමඟ පමණක් බවයි.
