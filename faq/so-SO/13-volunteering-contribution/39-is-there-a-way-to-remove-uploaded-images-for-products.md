Kaliya habeeyayaasha ayaa ka saari kara sawirrada, si looga fogaado burbur suurtagal ah.

Kaliya weydii Slack ama contact@openfoodfacts.org inay kaa saaraan nuqulladaada ama sawirrada aan habboonayn (waa inaad isku daydaa inaad bixiso lambarka barcode-ka ama URL-ka).

Waxaan sidoo kale haynaa API cusub oo warbixin sawireed ah haddii aad tahay barnaamij-sameeye.
