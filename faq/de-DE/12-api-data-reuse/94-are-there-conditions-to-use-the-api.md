Die gesamte Dokumentation zur API-Nutzung finden Sie auf der API-Dokumentationsseite. Hier eine kurze Zusammenfassung:

- Die Open Food Facts-Datenbank ist als offene Datenquelle unter der Open Database License (ODbL) verfügbar; die rechtlichen Details finden Sie unter https://world.openfoodfacts.org/terms-of-use. Die beiden Bedingungen sind Zuschreibung und Weitergabe unter gleichen Bedingungen. Wenn Sie Daten von Open Food Facts mit anderen Datenbanken kombinieren, verlangt das ODbL, dass die resultierende Datenbank ebenfalls als offene Daten veröffentlicht werden muss. Das bedeutet auch, dass Sie die Daten nur mit Quellen kombinieren können, die eine solche Weiterverteilung zulassen.

- Sie müssen bei API-Aufrufen stets einen benutzerdefinierten User-Agent verwenden, um Ihre App zu identifizieren.

- Für jeden API-Endpunkt gelten Ratenbegrenzungen.
