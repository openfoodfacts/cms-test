Open Food Facts conten pas qu'informacions sus la noiridura embalada. Per de valors mejanas de produches (per exemple, de tomatas o de bananas) e d'autres produches alimentaris, podètz utilizar una de las basas de donadas oficialas de nutricion nacionalas a la plaça.

Nota: La lista çai-jos conten qualques unas de las basas de donadas alimentarias nacionalas mai importantas. Se pensatz qu'una autra basa de donadas deuriá èsser inclusa dins la lista, contactatz-nos a: https://world.openfoodfacts.org/contact

Lista de las basas de donadas nacionalas de l'alimentacion

- Australia - FSANZ - NUTTAB 2006: https://www.foodstandards.gov.au/media/documents/FSANZ%20Conf%20PostersNUTTAB.pdf
- Belgica - NUBEL - Donadas de composicion alimentària bèlga: https://www.internubel.be
- Canadà - FCEN: https://aliments-nutrition.canada.ca/cnf-fce/index-fra.jsp
- Republica Chèca - Base de Donadas de Composicion Alimentària a l'Institut Nacional de Santat Publica: http://www.chpr.szu.cz/dbdata/foodcomp/nut2001.asp
- Dinamarca - Banca de donadas de la composicion alimentària danesa: https://frida.fooddata.dk/?lang=en
- Estònia - Base de donadas de composicion alimentària estoniana: https://tka.nutridata.ee/en/
- Finlàndia - Base de Donadas de Composicion Alimentària Finlandesa - FINELI: https://fineli.fi/fineli/en/index
- France - CIQUAL: https://www.anses.fr/en/search/site/Table%20ciqual
- Alemanha - Base de donadas en linha Souci-Fachmann-Kraut: https://www.sfk.online/#/home o la basa de donadas oficiala alemanda: Bundeslebensmittelschlüssel: https://blsdb.de/
- Italie - Banca Dati di Composizione degli Alimenti CREA: https://www.crea.gov.it/web/alimenti-e-nutrizione/banche-dati
- Païses Basses - Base de Donadas de Composicion Alimentària Neerlandesa: https://www.rivm.nl/en/base de donadas de composicion-alimentària neerlandesa
- Norvegia - La taula de la composicion alimentària norvegiana 2006: https://www.matvaretabellen.no/?language=en
- Polonha - Taulas de composicion alimentària: http://www.izz.waw.pl/en/?lang=en
- Espanha - Base de donadas de composicion alimentària espanhòla - BEDCA: https://www.bedca.net/bdpub/index.php
- Soïssa - Base de donadas de composicion alimentària soïssa: https://www.naehrwertdaten.ch/de/
- UK - Composition of foods integrated dataset (CoFID): https://www.gov.uk/government/publications/composition-of-foods-integrated-dataset-cofid
- USA - USDA: https://ndb.nal.usda.gov/
