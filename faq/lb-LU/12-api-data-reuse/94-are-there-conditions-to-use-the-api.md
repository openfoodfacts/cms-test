All Dokumentatioun iwwer d'Benotzung vun der API fannt Dir op der Säit mat der API-Dokumentatioun, awer hei ass eng kuerz Zesummefassung:

- D'Open Food Facts Datebank ass als Open Data ënner der Open Database License (ODbL) verfügbar, kuckt https://world.openfoodfacts.org/terms-of-use fir déi juristesch Detailer. Déi zwou Konditioune sinn Attributioun a Gläichberechtegung. Wann Dir Daten aus Open Food Facts mat aneren Datenbanken kombinéiert, dann verlaangt d'ODbL, datt déi resultéierend Datebank och als Open Data verëffentlecht muss ginn. Et bedeit och, datt Dir d'Donnéeën nëmme mat Quelle kombinéiere kënnt, déi eng sou eng Ëmverdeelung erlaben.

- Dir musst ëmmer e personaliséierten User-Agent benotzen, wann Dir API-Uruff ausféiert, fir Är App z'identifizéieren.

- Fir all API-Endpunkt ginn Tauxlimite duerchgesat.
