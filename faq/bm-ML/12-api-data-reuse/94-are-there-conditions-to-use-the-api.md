Sɛbɛn minnu bɛ API baarakɛcogo kan, olu bɛɛ bɛ se ka sɔrɔ API sɛbɛnni ɲɛ kan, nka nin ye a kuncɛlen teliya ye:

- Dumuni dafalenw kunnafonisɛbɛn bɛ sɔrɔ i n’a fɔ kunnafoni dafalenw Dakun dafalenw ka lase (ODbL) kɔnɔ, aw ye https://world.openfoodfacts.org/terms-of-use lajɛ walasa ka sariya kunnafoniw sɔrɔ. O sarati fla ye attribution (dafa) ni share-like (jatebɔ) ye. N’i ye kunnafoniw fara ɲɔgɔn kan ka bɔ Open Food Facts la ni kunnafonidilan wɛrɛw ye, o tuma na, ODbL b’a ɲini ko kunnafonidilan min bɛ sɔrɔ o la, o ka kan ka bɔ kɛnɛ kan i n’a fɔ kunnafonidilan dafalenw fana. O kɔrɔ fana ye ko i bɛ se ka kunnafoniw fara ɲɔgɔn kan dɔrɔn ni sɔrɔyɔrɔw ye minnu tun bɛna sira di o tila-tilacogo kura ma.

- I ka kan ka baara kɛ ni User-Agent ye tuma bɛɛ ni i bɛ API weleli kɛ walasa k’i ka appli dɔn.

- Rate-limits bɛ waleya API labanyɔrɔ kelen-kelen bɛɛ la.
