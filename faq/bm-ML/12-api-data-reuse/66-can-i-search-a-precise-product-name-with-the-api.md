A fɔ man di nka a ma Se fɔlɔ ka ɲinini Kɛ nɔgɔya la fɛn dilannen tɔgɔ dɔrɔn kan ani tigitigi API fɛ.

Baara kɛli ni filɛri ye min bɛ kɛ kulu kan, o bɛ se k’i dɛmɛ k’i ka ɲinini kɛ ka ɲɛ hali ni o kɛra.
