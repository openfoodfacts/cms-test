API Open Pet Food Facts на 98% супадае з API Open Food Facts, таму, калі вы карыстаецеся дакументацыяй API Open Food Facts, вы павінны мець даволі выразнае ўяўленне пра API Open Pet Food Facts.

Звярніце ўвагу, што такія функцыі, як панэлі ведаў, экалагічны бал і нутры-бал, не падтрымліваюцца (альбо з-за тэмы, альбо састарэлай кодавай базы).
