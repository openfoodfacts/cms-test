Усю дакументацыю па выкарыстанні API можна знайсці на старонцы дакументацыі API, але вось кароткі змест:

- База дадзеных Open Food Facts даступная ў выглядзе адкрытых дадзеных у адпаведнасці з ліцэнзіяй Open Database License (ODbL), юрыдычную інфармацыю глядзіце па спасылцы https://world.openfoodfacts.org/terms-of-use. Дзве ўмовы - гэта атрыбуцыя і распаўсюджванне на тых жа роўных умовах. Калі вы аб'ядноўваеце дадзеныя з Open Food Facts з іншымі базамі дадзеных, то ODbL патрабуе, каб атрыманая база дадзеных таксама была апублікавана як адкрытыя дадзеныя. Гэта таксама азначае, што вы можаце аб'ядноўваць дадзеныя толькі з крыніцамі, якія дазваляюць такое пераразмеркаванне.

- Пры выкананні выклікаў API для ідэнтыфікацыі вашага прыкладання вы заўсёды павінны выкарыстоўваць уласны агент карыстальніка.

- Абмежаванні хуткасці ўжываюцца для кожнай канчатковай кропкі API.
