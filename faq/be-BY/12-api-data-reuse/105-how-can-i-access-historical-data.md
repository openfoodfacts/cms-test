У цяперашні час мы не прапануем архіўныя дампы дадзеных (JSONL, MongoDB, CSV).

Аднак для асобных прадуктаў можна атрымаць доступ да папярэдніх версій дадзеных аб прадукце з дапамогай API або на старонцы прадукту з дапамогай рэдакцый.

Кожны раз, калі прадукт абнаўляецца, ствараецца новая рэдакцыя (павелічэнне лічбы пачынаючы з 1).

Напрыклад, каб атрымаць першую рэвізію (=першую версію прадукту) гэтага прадукту, выкарыстоўвайце

https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1.

Падобным чынам, параметр rev можна выкарыстоўваць з API:

https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1
