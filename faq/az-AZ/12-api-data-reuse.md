## Yüklənmiş fotoşəkillər üçün hansı ölçüdə yaxşı olacağına dair sənədlərdə hər hansı bir tövsiyə varmı?

Şəbəkənin yavaş və ya bahalı olması ölkələrdən asılı ola bilər. Çəki və ya boy baxımından 5000 pikseldən yuxarı olan hər hansı bir şey, ehtimal ki, çox faydalı deyil. Əgər şəbəkənin yavaş olduğunu bir şəkildə aşkar edə bilsəniz, hətta 2000 piksellik bir şəkil belə əla olardı (əlbəttə ki, şəklin olmamasından daha yaxşıdır!)

## Barkodsuz yeməklər necə olacaq?

Açıq Qida Faktları yalnız qablaşdırılmış qidalar haqqında məlumatları ehtiva edir. Məhsulların (məsələn, pomidor və ya banan) və digər qida məhsullarının orta dəyərləri üçün rəsmi milli qidalanma məlumat bazalarından birini istifadə edə bilərsiniz.

**Qeyd:** Aşağıdakı siyahıda ən vacib milli qida məlumat bazalarından bəziləri yer alır. Əgər siyahıya başqa bir verilənlər bazasının daxil edilməsini düşünürsünüzsə, zəhmət olmasa, bizimlə əlaqə saxlayın: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## API ilə dəqiq bir məhsul adını axtara bilərəmmi?

Təəssüf ki, yalnız məhsul adı ilə və dəqiq olaraq API vasitəsilə asanlıqla axtarış aparmaq hələ mümkün deyil.

Kateqoriya üzrə filtrdən istifadə etmək axtarışınızı daha dəqiq etməyə kömək edə bilər.

---

## Layihələrim üçün məlumatlara necə daxil ola/toplaya bilərəm?

Açıq Qida Faktları əsas səhifəsində, ekranın yuxarı sol küncündə sürüşdürmə menyusu var. Bunun altında, tıklaya biləcəyiniz "inkişaf etmiş axtarış" seçimini tapacaqsınız. Daha sonra layihəniz(ləriniz) üçün hansı meyarların ən uyğun olduğunu müəyyən etmək sizə qalıb. Seçildikdən sonra, səhifənin altındakı aşağıya fırladıb "Nəticələri yüklə" düyməsini basaraq əldə edilən nəticələri yükləyə biləcəksiniz.

Həmçinin məsləhətləşə bilərsiniz:

- API sənədlərimiz:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Açıq Qida Faktlarının İstifadə Şərtləri:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Məlumatlarımız haqqında:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## API-dən istifadə etmək üçün şərtlər varmı?

API istifadəsi ilə bağlı bütün sənədləri [API sənədləri səhifəsində](https://openfoodfacts.github.io/openfoodfacts-server/api/) tapa bilərsiniz, lakin qısa bir xülasə:

- Açıq Qida Faktları verilənlər bazası Açıq Verilənlər Bazası Lisenziyası (ODbL) çərçivəsində açıq məlumatlar kimi mövcuddur, hüquqi məlumatlar üçün [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) səhifəsinə baxın. İki şərt atribut və eyni paydır. Əgər Open Food Facts-dan məlumatları digər verilənlər bazaları ilə birləşdirsəniz, ODbL nəticədə əldə edilən verilənlər bazasının da açıq məlumat kimi yayımlanmasını tələb edir. Bu həm də o deməkdir ki, məlumatları yalnız bu cür yenidən bölüşdürməyə imkan verən mənbələrlə birləşdirə bilərsiniz.

- Tətbiqinizi müəyyən etmək üçün API çağırışları edərkən **həmişə** xüsusi İstifadəçi Agentindən istifadə etməlisiniz.

- Hər bir API son nöqtəsi üçün qiymət limitləri tətbiq olunur.

## Tarixi məlumatlara necə daxil ola bilərəm?

Hazırda tarixi məlumatların (JSONL, MongoDB, CSV) silinməsi təklif etmirik.

Lakin, fərdi məhsullar üçün məhsul məlumatlarının əvvəlki versiyalarına API istifadə edərək və ya məhsul səhifəsində düzəlişlərdən istifadə edərək daxil olmaq mümkündür.

Hər dəfə bir məhsul yeniləndikdə, yeni bir düzəliş (1-dən başlayaraq artan rəqəm) yaradılır.

Məsələn, bu məhsulun ilk versiyasını (=ilk məhsul versiyasını) əldə etmək üçün istifadə edin

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Eynilə, rev parametri API ilə istifadə edilə bilər:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
