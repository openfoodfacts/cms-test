## Gibt es irgendwo in der Dokumentation Empfehlungen, was eine gute Größe für hochgeladene Fotos wäre?

Das kann von den Ländern abhängen, wenn das Netz langsam oder teuer ist. Alles über 5000 Pixel in Gewicht oder Höhe ist wahrscheinlich nicht sehr nützlich. und wenn Sie irgendwie erkennen können, dass das Netz langsam ist, dann wäre sogar ein 2000 Pixel Bild großartig (sicherlich besser als kein Bild zu haben!)

## Was ist mit Lebensmitteln ohne Strichcode?

Open Food Facts enthält nur Informationen über verpackte Lebensmittel. Für Durchschnittswerte von Produkten (z. B. Tomaten oder Bananen) und anderen Lebensmitteln können Sie stattdessen eine der offiziellen nationalen Ernährungsdatenbanken verwenden.

  

Hinweis: Die folgende Liste enthält einige der wichtigsten nationalen Lebensmitteldatenbanken. Wenn Sie der Meinung sind, dass eine andere Datenbank in die Liste aufgenommen werden sollte, kontaktieren Sie uns bitte unter: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## Kann ich mit der API nach einem bestimmten Produktnamen suchen?

Leider ist es noch nicht möglich, über die API nur nach Produktnamen und präzise zu suchen.

  

Die Verwendung eines Filters für die Kategorie könnte Ihnen jedoch helfen, Ihre Suche zu präzisieren.

## Wie kann ich auf Daten für meine Projekte zugreifen bzw. sie sammeln?

Auf der Hauptseite von Open Food Facts befindet sich in der oberen linken Ecke des Bildschirms ein Rollmenü. Am unteren Ende des Menüs finden Sie die Option "Erweiterte Suche", auf die Sie klicken können. Sie können dann selbst bestimmen, welche Kriterien für Ihr(e) Projekt(e) am wichtigsten sind. Wenn Sie diese ausgewählt haben, können Sie die erhaltenen Ergebnisse herunterladen, indem Sie am Ende der Seite nach unten scrollen und auf "Ergebnisse herunterladen" klicken.

  

Sie können auch nachschlagen:

- Unsere API-Dokumentation:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Öffnen Sie die Nutzungsbedingungen von Food Facts:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Über unsere Daten: [https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)
