Auf der Hauptseite von Open Food Facts befindet sich in der oberen linken Ecke des Bildschirms ein Rollmenü. Am unteren Ende des Menüs finden Sie die Option "Erweiterte Suche", auf die Sie klicken können. Sie können dann selbst bestimmen, welche Kriterien für Ihr(e) Projekt(e) am wichtigsten sind. Wenn Sie diese ausgewählt haben, können Sie die erhaltenen Ergebnisse herunterladen, indem Sie am Ende der Seite nach unten scrollen und auf "Ergebnisse herunterladen" klicken.

Sie können auch nachschlagen:

- Unsere API-Dokumentation: https://openfoodfacts.github.io/api-documentation/
- Öffnen Sie die Nutzungsbedingungen von Food Facts: https://world.openfoodfacts.org/terms-of-use
- Über unsere Daten: https://world.openfoodfacts.org/data
