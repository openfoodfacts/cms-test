APIaren erabilerari buruzko dokumentazio guztia APIaren dokumentazio orrian aurki daiteke, baina hona hemen laburpen azkar bat:

- Open Food Facts datu-basea datu ireki gisa dago eskuragarri Open Database Lizentziaren (ODbL) pean; ikus https://world.openfoodfacts.org/terms-of-use xehetasun legalak lortzeko. Bi baldintzak atribuzioa eta berdin partekatzea dira. Open Food Facts-eko datuak beste datu-base batzuekin konbinatzen badituzu, ODbL-k eskatzen du emaitza den datu-basea ere datu ireki gisa argitaratu behar dela. Horrek esan nahi du datuak birbanaketa hori ahalbidetuko luketen iturriekin bakarrik konbina ditzakezula.

- API deiak egiterakoan, beti erabili behar duzu erabiltzaile-agente pertsonalizatu bat zure aplikazioa identifikatzeko.

- API amaiera-puntu bakoitzerako abiadura-mugak betearazten dira.
