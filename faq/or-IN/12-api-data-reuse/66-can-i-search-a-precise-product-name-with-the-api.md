ଦୁର୍ଭାଗ୍ୟବଶତଃ API ମାଧ୍ୟମରେ କେବଳ ଏବଂ ସଠିକ୍ ଭାବରେ ଉତ୍ପାଦ ନାମରେ ସହଜରେ ସନ୍ଧାନ କରିବା ଏପର୍ଯ୍ୟନ୍ତ ସମ୍ଭବ ହୋଇନାହିଁ।

ବର୍ଗରେ ଫିଲ୍ଟର ବ୍ୟବହାର କରିବା ଦ୍ଵାରା ଆପଣଙ୍କ ସନ୍ଧାନକୁ ଅଧିକ ସଠିକ୍ କରିବାରେ ସାହାଯ୍ୟ ହୋଇପାରେ।
