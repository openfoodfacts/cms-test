Całą dokumentację dotyczącą korzystania z interfejsu API można znaleźć na stronie dokumentacji API, ale oto jej krótkie podsumowanie:

- Baza danych Open Food Facts jest dostępna jako otwarte dane na podstawie licencji Open Database License (ODbL). Szczegóły prawne można znaleźć na stronie https://world.openfoodfacts.org/terms-of-use. Dwa warunki to atrybucja i udostępnianie na tych samych zasadach. Jeśli łączysz dane z Open Food Facts z danymi z innych baz danych, wówczas ODbL wymaga, aby powstała baza danych również została udostępniona jako otwarte dane. Oznacza to również, że łączenie danych możliwe jest wyłącznie ze źródłami, które dopuszczają taką redystrybucję.

- Podczas wywołań interfejsu API w celu identyfikacji aplikacji należy zawsze używać niestandardowego User-Agent.

- Limity przepustowości są egzekwowane dla każdego punktu końcowego interfejsu API.
