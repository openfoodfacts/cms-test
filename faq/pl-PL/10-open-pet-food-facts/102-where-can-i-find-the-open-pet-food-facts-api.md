Interfejs API Open Pet Food Facts jest w 98% taki sam jak interfejs API Open Food Facts, dlatego jeśli korzystasz z dokumentacji interfejsu API Open Food Facts, powinieneś mieć dość jasny obraz interfejsu API Open Pet Food Facts.

Należy pamiętać, że funkcje takie jak Panele wiedzy, Eco-Score, Nutri-Score nie są obsługiwane (ze względu na temat lub starą bazę kodu).

https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/
