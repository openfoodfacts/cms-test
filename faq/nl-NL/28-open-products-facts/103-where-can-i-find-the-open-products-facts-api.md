De Open Products Facts API is voor 98% hetzelfde als de Open Food Facts API, dus als je de documentatie van de Open Food Facts API gebruikt, krijg je een vrij duidelijk beeld van de Open Products Facts API. We hebben ook een speciale handleiding geschreven waarmee u universeel scannen in uw app kunt implementeren.

https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/
