Open Pet Food Facts API သည် Open Food Facts API နှင့် 98% တူညီသောကြောင့် Open Food Facts API စာရွက်စာတမ်းကို အသုံးပြုပါက Open Pet Food Facts API ကို ရှင်းရှင်းလင်းလင်း မြင်ရပါမည်။

Knowledge Panels၊ Eco-Score၊ Nutri-Score ကဲ့သို့သော အင်္ဂါရပ်များကို မပံ့ပိုးကြောင်း သတိပြုပါ (ခေါင်းစဉ်ကြောင့်ဖြစ်စေ၊ ကုဒ်အခြေခံအဟောင်းကြောင့်ဖြစ်စေ)။
