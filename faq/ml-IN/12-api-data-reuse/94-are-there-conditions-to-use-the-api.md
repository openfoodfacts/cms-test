API ഉപയോഗത്തെക്കുറിച്ചുള്ള എല്ലാ ഡോക്യുമെന്റേഷനുകളും API ഡോക്യുമെന്റേഷൻ പേജിൽ കാണാം, എന്നാൽ ഇതാ ഒരു ചെറിയ സംഗ്രഹം:

- ഓപ്പൺ ഡാറ്റാബേസ് ലൈസൻസിന് (ODbL) കീഴിൽ ഓപ്പൺ ഫുഡ് ഫാക്റ്റ്സ് ഡാറ്റാബേസ് ഓപ്പൺ ഡാറ്റയായി ലഭ്യമാണ്, നിയമപരമായ വിശദാംശങ്ങൾക്ക് https://world.openfoodfacts.org/terms-of-use കാണുക. ആട്രിബ്യൂഷനും ഷെയറും ഒരുപോലെയാണ് രണ്ട് വ്യവസ്ഥകളും. ഓപ്പൺ ഫുഡ് ഫാക്‌ട്‌സിൽ നിന്നുള്ള ഡാറ്റ മറ്റ് ഡാറ്റാബേസുകളുമായി സംയോജിപ്പിക്കുകയാണെങ്കിൽ, തത്ഫലമായുണ്ടാകുന്ന ഡാറ്റാബേസ് ഓപ്പൺ ഡാറ്റയായി പുറത്തുവിടണമെന്ന് ODbL ആവശ്യപ്പെടുന്നു. അതായത്, പുനർവിതരണം അനുവദിക്കുന്ന ഉറവിടങ്ങളുമായി മാത്രമേ നിങ്ങൾക്ക് ഡാറ്റ സംയോജിപ്പിക്കാൻ കഴിയൂ.

- നിങ്ങളുടെ ആപ്പ് തിരിച്ചറിയാൻ API കോളുകൾ ചെയ്യുമ്പോൾ എപ്പോഴും ഒരു ഇഷ്ടാനുസൃത യൂസർ-ഏജന്റ് ഉപയോഗിക്കണം.

- ഓരോ API എൻഡ്‌പോയിന്റിനും നിരക്ക് പരിധികൾ നടപ്പിലാക്കിയിരിക്കുന്നു.
