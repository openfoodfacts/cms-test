നിർഭാഗ്യവശാൽ ഉൽപ്പന്നത്തിന്റെ പേരിൽ മാത്രം എളുപ്പത്തിൽ തിരയാനും API വഴി കൃത്യമായി തിരയാനും ഇതുവരെ സാധ്യമല്ല.

വിഭാഗത്തിൽ ഒരു ഫിൽട്ടർ ഉപയോഗിക്കുന്നത് നിങ്ങളുടെ തിരയൽ കൂടുതൽ കൃത്യമാക്കാൻ സഹായിച്ചേക്കാം.
