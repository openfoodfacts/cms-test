ഓപ്പൺ ഫുഡ് ഫാക്ട്സ് API, ഓപ്പൺ ബ്യൂട്ടി ഫാക്ട്സ് API-യുടെ 98% സമാനമാണ്, അതിനാൽ നിങ്ങൾ ഓപ്പൺ ഫുഡ് ഫാക്ട്സ് API ഡോക്യുമെന്റേഷൻ ഉപയോഗിക്കുകയാണെങ്കിൽ ഓപ്പൺ ബ്യൂട്ടി ഫാക്ട്സ് API-യെക്കുറിച്ച് നിങ്ങൾക്ക് വ്യക്തമായ ഒരു ധാരണ ഉണ്ടായിരിക്കണം.

നോളജ് പാനലുകൾ, ഇക്കോ-സ്കോർ, ന്യൂട്രി-സ്കോർ തുടങ്ങിയ സവിശേഷതകൾ പിന്തുണയ്ക്കുന്നില്ല (വിഷയം കാരണമോ പഴയ കോഡ്ബേസ് കാരണമോ).
