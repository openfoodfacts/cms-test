L'API Open Products Facts est identique à 98 % à l'API Open Food Facts ; par conséquent, si vous utilisez la documentation de l'API Open Food Facts, vous devriez avoir une vision assez claire de l'API Open Products Facts. Nous avons également rédigé un tutoriel dédié pour vous permettre d'implémenter la numérisation universelle dans votre application.

https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/
