Nous utilisons le code-barres pour identifier les produits. Si vous importez un enregistrement avec le même code-barres qu'un enregistrement existant, les informations seront fusionnées.

Pour les données telles que la liste des ingrédients et les valeurs nutritionnelles (une seule valeur correcte est possible), les données envoyées via la plateforme du producteur écraseront les données existantes.

Pour les données telles que les étiquettes/catégories/marques (plusieurs valeurs correctes possibles), les informations sont fusionnées. Si les données sont incorrectes sur la plateforme publique, vous pouvez modifier la fiche produit sur la plateforme publique afin de supprimer les valeurs incorrectes.
