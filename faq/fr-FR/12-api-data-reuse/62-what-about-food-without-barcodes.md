Open Food Facts ne contient que des informations sur les aliments emballés. Pour connaître les valeurs moyennes des fruits et légumes (par exemple, des tomates ou des bananes) et autres produits alimentaires, vous pouvez utiliser l'une des bases de données nutritionnelles nationales officielles.

Remarque : La liste ci-dessous contient quelques-unes des bases de données alimentaires nationales les plus importantes. Si vous pensez qu'une autre base de données devrait figurer dans la liste, veuillez nous contacter à l'adresse suivante : https://world.openfoodfacts.org/contact

Liste des bases de données alimentaires nationales

- Australie - FSANZ - NUTTAB 2006 : https://www.foodstandards.gov.au/media/documents/FSANZ%20Conf%20PostersNUTTAB.pdf
- Belgique - NUBEL - Données belges sur la composition des aliments : https://www.internubel.be
- Canada - FCEN : https://aliments-nutrition.canada.ca/cnf-fce/index-fra.jsp
- République tchèque - Base de données sur la composition des aliments de l'Institut national de santé publique : http://www.chpr.szu.cz/dbdata/foodcomp/nut2001.asp
- Danemark - Base de données danoise sur la composition des aliments : https://frida.fooddata.dk/?lang=en
- Estonie - Base de données estonienne sur la composition des aliments : https://tka.nutridata.ee/en/
- Finlande - Base de données finlandaise sur la composition des aliments - FINELI : https://fineli.fi/fineli/en/index
- France - CIQUAL : https://www.anses.fr/en/search/site/Table%20ciqual
- Allemagne - Base de données en ligne Souci-Fachmann-Kraut : https://www.sfk.online/#/home ou la base de données officielle allemande : Bundeslebensmittelschlüssel : https://blsdb.de/
- Italie - Banca Dati di Composizione degli Alimenti CREA : https://www.crea.gov.it/web/alimenti-e-nutrizione/banche-dati
- Pays-Bas - Base de données néerlandaise sur la composition des aliments : https://www.rivm.nl/en/dutch-food-composition-database
- Norvège - Tableau de composition des aliments norvégiens 2006 : https://www.matvaretabellen.no/?language=en
- Pologne - Tables de composition des aliments : http://www.izz.waw.pl/en/?lang=en
- Espagne - Base de données espagnole sur la composition des aliments - BEDCA : https://www.bedca.net/bdpub/index.php
- Suisse - Base de données suisse sur la composition des aliments : https://www.naehrwertdaten.ch/de/
- Royaume-Uni - Base de données intégrée sur la composition des aliments (CoFID) : https://www.gov.uk/government/publications/composition-of-foods-integrated-dataset-cofid
- États-Unis - USDA : https://ndb.nal.usda.gov/
