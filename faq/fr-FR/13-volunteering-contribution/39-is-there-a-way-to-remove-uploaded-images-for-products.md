---
title: "Existe-t-il un moyen de supprimer les images téléchargées pour les produits ?"
order: 39
lang: en-gb
category: 13-contribution de bénévolat
breadcrumbs: [ '/en-gb/', '/fr-fr/13-contribution-volontaire/' ]

Seuls les modérateurs peuvent supprimer les photos, afin d'éviter tout risque de vandalisme.

Il suffit de demander sur Slack ou à contact@openfoodfacts.org de supprimer vos doublons ou toute photo inappropriée (vous devrez essayer de fournir le numéro de code-barres ou l'URL à cet effet).

Nous proposons également une nouvelle API de rapports d'images si vous êtes programmeur.
