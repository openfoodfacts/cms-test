## ስለምንታይ ንእንስሳታት ዝኸውን ኒውትሪ-ስኮር ዘይብሉ?

ነፍሲ ወከፍ እንስሳ ዝተፈላለየ እዩ (ተመን ከም ድሙ ወይ ከልቢ ተመሳሳሊ ድሌት መግቢ የብሉን)

ኣብ ውሽጢ ሓደ ዓይነት እንስሳ ከም ኣኽላባት እውን እንተኾነ፡ እቲ ድሌታት ኣመጋግባ ከከም ህይወት ናይቲ ስቕ ዝበለ ኣነባብራ፡ ሓደ ክኸውን ዝኽእል ምቑራጽ ወይ ሕማማት ይፈላለ እዩ።

ስለዚ ንዓሰርተታት ተመራመርቲ ከየሳተፍካን ዓሰርተታት ዝተፈላለዩ ኣልጎሪዝማት ከይፈጠርካን ንመግቢ እንስሳታት ነጥቢ ምሕሳብ ዝተሓላለኸ ኮይኑ፡ ክሳብ ሕጂ ዘይተገብረ እዩ።

## Where can I find the Open Pet Food Facts API ?

Open Pet Food Facts API is 98% the same as Open Food Facts API so if you use Open Food Facts API documentation you should have a pretty clear view of the Open Pet Food Facts API.

Note that features like the Knowledge Panels, Eco-Score, Nutri-Score are not supported (either due to the topic, or to the old codebase).

[https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/](https://ክፉት ምግቢ ሓቅታት.github.io/ክፉት ምግቢ-ኣገልጋሊ/api/ref-v2/)
