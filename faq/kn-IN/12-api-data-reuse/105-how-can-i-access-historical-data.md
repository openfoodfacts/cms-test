ಪ್ರಸ್ತುತ, ನಾವು ಐತಿಹಾಸಿಕ ಡೇಟಾ ಡಂಪ್ (JSONL, MongoDB, CSV) ಅನ್ನು ನೀಡುವುದಿಲ್ಲ.

ಆದಾಗ್ಯೂ, ಪ್ರತ್ಯೇಕ ಉತ್ಪನ್ನಗಳಿಗೆ, API ಬಳಸಿಕೊಂಡು ಅಥವಾ ಉತ್ಪನ್ನ ಪುಟದಲ್ಲಿ ಪರಿಷ್ಕರಣೆಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಉತ್ಪನ್ನ ಡೇಟಾದ ಹಿಂದಿನ ಆವೃತ್ತಿಗಳನ್ನು ಪ್ರವೇಶಿಸಲು ಸಾಧ್ಯವಿದೆ.

ಪ್ರತಿ ಬಾರಿ ಉತ್ಪನ್ನವನ್ನು ನವೀಕರಿಸಿದಾಗ, ಹೊಸ ಪರಿಷ್ಕರಣೆ (1 ರಿಂದ ಪ್ರಾರಂಭವಾಗುವ ಹೆಚ್ಚುತ್ತಿರುವ ಅಂಕೆ) ರಚಿಸಲ್ಪಡುತ್ತದೆ.

ಉದಾಹರಣೆಗೆ, ಈ ಉತ್ಪನ್ನದ ಮೊದಲ ಪರಿಷ್ಕರಣೆ (=ಮೊದಲ ಉತ್ಪನ್ನ ಆವೃತ್ತಿ) ಪಡೆಯಲು, ಬಳಸಿ

https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1.
