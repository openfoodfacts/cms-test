ದುರದೃಷ್ಟವಶಾತ್ ಉತ್ಪನ್ನದ ಹೆಸರಿನಲ್ಲಿ ಮಾತ್ರ ಸುಲಭವಾಗಿ ಮತ್ತು ನಿಖರವಾಗಿ API ಮೂಲಕ ಹುಡುಕಲು ಇನ್ನೂ ಸಾಧ್ಯವಾಗಿಲ್ಲ.

ವರ್ಗದಲ್ಲಿ ಫಿಲ್ಟರ್ ಬಳಸುವುದರಿಂದ ನಿಮ್ಮ ಹುಡುಕಾಟವನ್ನು ಹೆಚ್ಚು ನಿಖರವಾಗಿ ಮಾಡಲು ಸಹಾಯವಾಗುತ್ತದೆ.
