Open Pet Food Facts API интерфейсі Open Food Facts API интерфейсімен 98% бірдей, сондықтан егер сіз Open Food Facts API құжаттамасын пайдалансаңыз, сізде Open Pet Food Facts API интерфейсі туралы түсінікті түсінік болуы керек.

Білім панельдері, Eco-Score, Nutri-Score сияқты мүмкіндіктерге қолдау көрсетілмейтінін ескеріңіз (тақырыпқа немесе ескі код базасына байланысты).
