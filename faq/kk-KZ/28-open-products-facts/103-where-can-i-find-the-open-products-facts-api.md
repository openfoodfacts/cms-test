Open Products Facts API интерфейсі Open Food Facts API интерфейсімен 98% бірдей, сондықтан егер сіз Open Food Facts API құжаттамасын пайдалансаңыз, Open Products Facts API интерфейсі туралы түсінікті болуы керек. Сондай-ақ, біз сіздің қолданбаңызда әмбебап сканерлеуді енгізуге мүмкіндік беретін арнайы оқулық жаздық.
