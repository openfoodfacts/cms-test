Open Food Facts API интерфейсі Open Beauty Facts API интерфейсімен 98% бірдей, сондықтан егер сіз Open Food Facts API құжаттамасын пайдалансаңыз, Open Beauty Facts API интерфейсі туралы түсінікті болуы керек.

Білім панельдері, Eco-Score, Nutri-Score сияқты мүмкіндіктерге қолдау көрсетілмейтінін ескеріңіз (тақырыпқа немесе ескі код базасына байланысты).
