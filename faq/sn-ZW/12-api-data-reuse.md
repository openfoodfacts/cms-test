## Ko nezvechikafu chisina mabarcode?

## Ndingatsvaga zita chairo rechigadzirwa neAPI here?

Zvinosuwisa kuti hazvigoneke kutsvaga zviri nyore pazita rechigadzirwa chete uye nemazvo kuburikidza neAPI.

Kushandisa sefa pane chikamu kunogona kukubatsira kuti kutsvaga kwako kuve kwakanyatsojeka.

## Ndingawana sei/kuunganidza sei data remapurojekiti angu?

Papeji guru reOpen Food Facts, pakona yepamusoro kuruboshwe rwechidzitiro, pane menyu yekuskorora. Pazasi payo, uchawana sarudzo ye "kutsvaga kwakadzama", yaunogona kudzvanya. Zvino zvave kwauri kusarudza kuti ndezvipi zvinodiwa zvakanyanya pabasa rako. Kana wangosarudza, unokwanisa kudhawunirodha zvawanikwa nekuskira pasi pazasi peji wodzvanya pakanzi “Dhawunirodha zvabuda”.

Unogonawo kubvunza:

- Magwaro edu eAPI:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Mitemo Yekushandisa yeOpen Food Facts:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)

## Pane mamiriro ekushandisa API here?

Magwaro ese nezvekushandiswa kweAPI anogona kuwanikwa pa [API documentation page](https://openfoodfacts.github.io/openfoodfacts-server/api/), asi heino pfupiso pfupi:

- Dhatabhesi reOpen Food Facts rinowanikwa sedata rakavhurika pasi peOpen Database License (ODbL), ona [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) kuti uwane ruzivo rwepamutemo. Mamiriro maviri aya ndeekuti kugoverwa (attribution) uye kugovana zvakafanana (share-like). Kana ukabatanidza data kubva kuOpen Food Facts nemamwe madhatabhesi, saka ODbL inoda kuti dhatabhesi rinobuda ribudiswe sedata rakavhurika zvakare. Zvinorevawo kuti unogona kusanganisa data chete nezvinyorwa zvinobvumira kugoverwa kwakadaro.

- Unofanira kushandisa **nguva dzose** User-Agent yakagadzirirwa iwe paunenge uchiita ma API calls kuti uzive app yako.

- Miganho yemitengo inomanikidzwa kune yega yega API endpoint.

## Ndingawana sei ruzivo rwenhoroondo?

Parizvino, hatipe data rekare (JSONL, MongoDB, CSV).

Zvisinei, kune zvigadzirwa zvega zvega, zvinokwanisika kuwana shanduro dzekare dzedata rechigadzirwa uchishandisa API kana papeji rechigadzirwa uchishandisa magadzirirwo.

Nguva imwe neimwe chigadzirwa chikagadziridzwa, panobva pagadzirwa imwe itsva (inotangira pa1) inovandudzwa.
