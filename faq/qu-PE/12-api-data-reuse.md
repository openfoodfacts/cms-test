## ¿Maypipas qillqakunapi yuyaychaykuna kanchu ima allin sayayniyuq kanman kargasqa fotokunapaq?

Chayqa suyukunamanta kanman, sichus llika pisilla utaq ancha chaninniyuq. imapas 5000 pixelmanta aswan llasayninpi icha sayayninpipas manapaschá ancha allinchu. chaymanta sichus imaynatapas chay llika llasaq kayninta riqsiyta atikunki, chaymanta huk 2000 pixel siq'ipas ancha allinmi kanman (chiqamanta aswan allin mana siq'iyuq kaymanta!) .

## ¿Imaynatataq proyectoykunapaq datosman yaykuyman/huñuyman?

Mikhuy Chiqakuna Kichasqa hatun p'anqapi, pantallap paña pata k'uchunpi, huk kuyuriq menú kachkan. Chaypa urayninpim tarinki “awpaq maskay” nisqa akllanata, chaypim ñitiyta atinki. Chaymantaqa qammanta kanqa mayqin criteriokuna aswan relevante kanku proyectoykipaq (kuna). Huk kuti akllasqa, atikunki uraykachiyta tarisqa ruwaykunata urayman kuyurispa urayman p’anqapi chaymanta ñit’ispa “Uraykachiy ruwasqakunata”.

Hinallataqmi tapuyta atinki:

- API qillqayku:[ https://openfoodfacts.github.io/api-documentation/](https://qu.wiqasqa mikhuykuna.github.io/api-documentation/)
- Kichasqa Mikhuy Chiqakuna' Uyway Términos:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/términos-de-uso)
- Willayniykumanta:[ https://world.openfoodfacts.org/data](https://qu.wich.openfoodfacts.org/datos)

## ¿API llamk'achinapaq condicionkuna kanchu?

API llamk'achiyta llapa qillqakuna [API qillqa p'anqapi](https://openfoodfacts.github.io/openfoodfacts-server/api/) nisqapi tarikunman, ichataq kaypi utqaylla p'anqa:

- Kichasqa Mikhuy Chiqap willakuypa qullqisapan kichasqa willakuy hina Kichasqa Willaypa Wasin Licencia (ODbL) kaqpi tarikun, qhaway [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) kamachiy willakuykunapaq. Iskay condicionkunaqa atribución nisqawan compartición nisqawanmi kaqlla. Sichus Kichasqa Mikhuy Chiqakunamanta willayta wak willaypa tiyapuyninwan tinkinki, chaymanta ODbL mañan chaymanta ruwasqa willaypa tiyapuynin kichasqa willay hina kacharisqa kanan tiyan. Chantapas niyta munan, willayta chayhina wakmanta rakiyta saqinman pukyukunawanlla tinkiyta atikunki.

- **sapa kuti** huk sapanchasqa Usuario-Agente llamk'achinayki tiyan mayk'aq API waqyakuykunata ruwachkanki ruwanayki riqsichinapaq.

- Tarifa-limitakuna sapa API tukukuypaq kamachisqa kanku.
