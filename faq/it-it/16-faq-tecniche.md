## Devo aggiornare tutti i file di lingua quando modifico una stringa sorgente?

No, non è così. È sufficiente aggiornare quello in inglese

- Crea il tuo PR

Una volta unito, il sistema di traduzione di Crowdin, attivato dalle azioni di GitHub, farà il resto per le altre lingue.

Il bot di GitHub crea quindi automaticamente una nuova PR che noi poi revisioniamo.
