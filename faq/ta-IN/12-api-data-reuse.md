## பதிவேற்றப்பட்ட புகைப்படங்களுக்கு எந்த அளவு நல்லதாக இருக்கும் என்பது குறித்து ஆவணத்தில் எங்காவது பரிந்துரைகள் உள்ளதா?

நெட்வொர்க் மெதுவாகவோ அல்லது விலை அதிகமாகவோ இருந்தால் அது நாடுகளைப் பொறுத்தது. 5000 பிக்சல்களுக்கு மேல் எடை அல்லது உயரம் உள்ள எதுவும் மிகவும் பயனுள்ளதாக இருக்காது. நெட்வொர்க் மெதுவாக இருப்பதை எப்படியாவது கண்டறிய முடிந்தால், 2000 பிக்சல்கள் கொண்ட படம் கூட சிறப்பாக இருக்கும் (நிச்சயமாக படம் இல்லாததை விட சிறந்தது!)

## பார்கோடுகள் இல்லாத உணவு எப்படி இருக்கும்?

திறந்த உணவு உண்மைகள் தொகுப்பில் தொகுக்கப்பட்ட உணவு பற்றிய தகவல்கள் மட்டுமே உள்ளன. விளைபொருட்களின் சராசரி மதிப்புகளுக்கு (உதாரணமாக, தக்காளி அல்லது வாழைப்பழங்கள்) மற்றும் பிற உணவுப் பொருட்களுக்கு, நீங்கள் அதிகாரப்பூர்வ தேசிய ஊட்டச்சத்து தரவுத்தளங்களில் ஒன்றைப் பயன்படுத்தலாம்.

**குறிப்பு:** கீழே உள்ள பட்டியலில் மிக முக்கியமான தேசிய உணவு தரவுத்தளங்கள் சில உள்ளன. பட்டியலில் வேறு ஏதேனும் தரவுத்தளம் சேர்க்கப்பட வேண்டும் என்று நீங்கள் நினைத்தால், தயவுசெய்து எங்களை இங்கே தொடர்பு கொள்ளவும்: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## API ஐப் பயன்படுத்தி ஒரு துல்லியமான தயாரிப்பு பெயரை நான் தேட முடியுமா?

துரதிர்ஷ்டவசமாக, தயாரிப்பு பெயரில் மட்டும் எளிதாகவும் துல்லியமாகவும் API மூலம் தேடுவது இன்னும் சாத்தியமில்லை.

வகையின் அடிப்படையில் ஒரு வடிப்பானைப் பயன்படுத்துவது உங்கள் தேடலை இன்னும் துல்லியமாக்க உதவும்.

## எனது திட்டங்களுக்கான தரவை எவ்வாறு அணுகுவது/சேகரிப்பது?

ஓபன் ஃபுட் ஃபேக்ட்ஸின் பிரதான பக்கத்தில், திரையின் மேல் இடது மூலையில், ஒரு ஸ்க்ரோலிங் மெனு உள்ளது. அதன் கீழே, நீங்கள் "மேம்பட்ட தேடல்" விருப்பத்தைக் காண்பீர்கள், அதை நீங்கள் கிளிக் செய்யலாம். உங்கள் திட்டத்திற்கு(களுக்கு) எந்த அளவுகோல்கள் மிகவும் பொருத்தமானவை என்பதைத் தீர்மானிப்பது உங்களுடையது. தேர்வு செய்தவுடன், பக்கத்தின் கீழே உருட்டி "முடிவுகளைப் பதிவிறக்கு" என்பதைக் கிளிக் செய்வதன் மூலம் பெறப்பட்ட முடிவுகளைப் பதிவிறக்க முடியும்.

நீங்கள் இவற்றையும் கலந்தாலோசிக்கலாம்:

- எங்கள் API ஆவணங்கள்:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- திறந்த உணவு உண்மைகளின் பயன்பாட்டு விதிமுறைகள்:[ ](>https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- எங்கள் தரவுகளில்:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## API-ஐப் பயன்படுத்த ஏதேனும் நிபந்தனைகள் உள்ளதா?

API பயன்பாடு பற்றிய அனைத்து ஆவணங்களையும் [API ஆவணப் பக்கத்தில்](https://openfoodfacts.github.io/openfoodfacts-server/api/காணலாம்), ஆனால் இங்கே ஒரு விரைவான சுருக்கம் உள்ளது:

- திறந்த உணவு உண்மைகள் தரவுத்தளம் திறந்த தரவுத்தள உரிமத்தின் (ODbL) கீழ் திறந்த தரவாகக் கிடைக்கிறது, சட்ட விவரங்களுக்கு [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) ஐப் பார்க்கவும். இரண்டு நிபந்தனைகளும் பண்புக்கூறு மற்றும் பங்கு-ஒரே மாதிரியானவை. திறந்த உணவு உண்மைகளிலிருந்து தரவை மற்ற தரவுத்தளங்களுடன் இணைத்தால், அதன் விளைவாக வரும் தரவுத்தளத்தையும் திறந்த தரவாக வெளியிட வேண்டும் என்று ODbL கோருகிறது. மேலும், மறுபகிர்வை அனுமதிக்கும் மூலங்களுடன் மட்டுமே நீங்கள் தரவை இணைக்க முடியும் என்பதையும் இது குறிக்கிறது.

- உங்கள் பயன்பாட்டை அடையாளம் காண API அழைப்புகளைச் செய்யும்போது நீங்கள் **எப்போதும்** தனிப்பயன் பயனர் முகவரைப் பயன்படுத்த வேண்டும்.

- ஒவ்வொரு API இறுதிப்புள்ளிக்கும் விகித வரம்புகள் செயல்படுத்தப்படுகின்றன.

## வரலாற்றுத் தரவை நான் எவ்வாறு அணுகுவது?

தற்போது, நாங்கள் வரலாற்று தரவு டம்பை (JSONL, MongoDB, CSV) வழங்குவதில்லை.

இருப்பினும், தனிப்பட்ட தயாரிப்புகளுக்கு, API ஐப் பயன்படுத்தி அல்லது திருத்தங்களைப் பயன்படுத்தி தயாரிப்பு பக்கத்தில் தயாரிப்பு தரவின் முந்தைய பதிப்புகளை அணுக முடியும்.

ஒரு தயாரிப்பு புதுப்பிக்கப்படும் ஒவ்வொரு முறையும், ஒரு புதிய திருத்தம் (1 இலிருந்து தொடங்கும் அதிகரிக்கும் இலக்கம்) உருவாக்கப்படுகிறது.

உதாரணமாக, இந்த தயாரிப்பின் முதல் திருத்தத்தை (=முதல் தயாரிப்பு பதிப்பு) பெற, பயன்படுத்தவும்

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).
