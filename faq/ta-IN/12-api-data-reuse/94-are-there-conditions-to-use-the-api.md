API பயன்பாடு பற்றிய அனைத்து ஆவணங்களையும் API ஆவணப் பக்கத்தில் காணலாம், ஆனால் இங்கே ஒரு விரைவான சுருக்கம் உள்ளது:

- திறந்த உணவு உண்மைகள் தரவுத்தளம் திறந்த தரவுத்தள உரிமத்தின் (ODbL) கீழ் திறந்த தரவாகக் கிடைக்கிறது, சட்ட விவரங்களுக்கு https://world.openfoodfacts.org/terms-of-use ஐப் பார்க்கவும். இரண்டு நிபந்தனைகளும் பண்புக்கூறு மற்றும் பங்கு-ஒரே மாதிரியானவை. திறந்த உணவு உண்மைகளிலிருந்து தரவை மற்ற தரவுத்தளங்களுடன் இணைத்தால், அதன் விளைவாக வரும் தரவுத்தளத்தையும் திறந்த தரவாக வெளியிட வேண்டும் என்று ODbL கோருகிறது. மேலும், மறுபகிர்வை அனுமதிக்கும் மூலங்களுடன் மட்டுமே நீங்கள் தரவை இணைக்க முடியும் என்பதையும் இது குறிக்கிறது.

- உங்கள் பயன்பாட்டை அடையாளம் காண API அழைப்புகளைச் செய்யும்போது எப்போதும் தனிப்பயன் பயனர் முகவரைப் பயன்படுத்த வேண்டும்.

- ஒவ்வொரு API இறுதிப்புள்ளிக்கும் விகித வரம்புகள் செயல்படுத்தப்படுகின்றன.
