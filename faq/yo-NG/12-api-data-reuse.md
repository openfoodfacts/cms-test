## Àwọn oúnjẹ tí kò ní àmì ìdámọ̀ràn ńkọ́?

Àwọn Ìròyìn nípa oúnjẹ tí a fi sínú àpótí nìkan ló wà nínú Ìwé Ìròyìn Oúnjẹ. Fún iye apapọ ti awọn irugbin (fun apẹẹrẹ, awọn tomati tabi awọn ọgẹmọ) ati awọn ọja ounjẹ miiran, o le lo ọkan ninu awọn ibi ipamọ data ounjẹ orilẹ-ede ti o jẹ aṣẹ dipo.

**Àkíyèsí:** Àkójọ tó wà ní ìsàlẹ̀ yìí ní díẹ̀ lára àwọn ibi ìkópamọ́ oúnjẹ orílẹ̀-èdè tó ṣe pàtàkì jùlọ. Tí o bá rò pé ó yẹ kí a fi ibi ìpamọ́ dátà míràn kún àkójọ náà, jọ̀wọ́ kàn sí wa ní: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## Báwo ni mo ṣe le wọlé/kó àwọn dátà jọ fún àwọn iṣẹ́ àgbékalẹ̀ mi?

Ní ojú ìwé àkọ́kọ́ Open Food Facts, ní igun òsì òkè ti ìbòjú náà, àkójọ ìyípo kan wà. Ní ìsàlẹ̀ rẹ̀, ìwọ yóò rí àṣàyàn “ìwádìí tó ti ní ìlọsíwájú”, èyí tí o lè tẹ̀ lórí rẹ̀. Lẹ́yìn náà, ó kù sí ọwọ́ rẹ láti pinnu àwọn ìlànà tí ó ṣe pàtàkì jùlọ fún iṣẹ́(àwọn) iṣẹ́ rẹ. Nígbà tí o bá ti yan èyí, o lè gba àwọn àbájáde tí o rí nípa yíyí lọ sí ìsàlẹ̀ ojú ìwé náà kí o sì tẹ “Gbigba àwọn àbájáde” lórí ìkànnì ayélujára.

O tun le kan si:

- Àwọn ìwé API wa:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Àwọn Ìlànà Lílo Àwọn Ìròyìn Oúnjẹ Ṣíṣí:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Lórí dátà wa:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## Ǹjẹ́ àwọn àdéhùn kan wà láti lo API náà?

Gbogbo ìwé nípa lílo API ni a lè rí lórí [ojú ìwé API documentary page](https://openfoodfacts.github.io/openfoodfacts-server/api/), ṣùgbọ́n àkópọ̀ kúkúrú kan nìyí:

- Ibi ipamọ data Open Food Facts wa bi data ṣiṣi labẹ Iwe-aṣẹ Open Database (ODbL), wo [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) fun awọn alaye ofin. Àwọn ipò méjèèjì ni ìfàmọ́ra àti ìpín-bákan náà. Tí o bá so dátà láti Open Food Facts pọ̀ mọ́ àwọn dátà ìpamọ́ mìíràn, nígbà náà ODbL nílò pé kí a tú dátà ìpamọ́ tí ó jáde jáde gẹ́gẹ́ bí dátà tí ó ṣí sílẹ̀ pẹ̀lú. Ó tún túmọ̀ sí wípé o lè so dátà pọ̀ mọ́ àwọn orísun tí yóò gba irú àtúnpín bẹ́ẹ̀ láàyè.

- O gbọ́dọ̀ lo Aṣojú Olùlò-àṣà kan nígbà gbogbo nígbà tí o bá ń ṣe àwọn ìpè API láti dá àpù rẹ mọ̀.

- A fi opin si oṣuwọn fun opin API kọọkan.

## Báwo ni mo ṣe le wọle si awọn data itan?

Lọ́wọ́lọ́wọ́, a kò ní àwọn ohun èlò ìforúkọsílẹ̀ ìtàn (JSONL, MongoDB, CSV).

Sibẹsibẹ, fun awọn ọja kọọkan, o ṣee ṣe lati wọle si awọn ẹya ti tẹlẹ ti data ọja nipa lilo API tabi lori oju-iwe ọja nipa lilo awọn atunṣe.

Nígbàkúgbà tí a bá ṣe àtúnṣe ọjà kan, àtúnṣe tuntun kan (tí ó ń pọ̀ sí i láti 1) ni a máa ń ṣẹ̀dá.

Fún àpẹẹrẹ, láti gba àtúnṣe àkọ́kọ́ (=àtúnṣe ọjà àkọ́kọ́) ti ọjà yìí, lo

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Bákan náà, a lè lo àtúnṣe rev pẹ̀lú API:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
