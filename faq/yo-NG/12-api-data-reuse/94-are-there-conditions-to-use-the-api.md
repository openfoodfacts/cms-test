Gbogbo ìwé nípa lílo API ni a lè rí lórí ojú ìwé API, ṣùgbọ́n àkópọ̀ kúkúrú kan nìyí:

- Ibi ipamọ data Ounjẹ Open Facts wa bi data ṣiṣi labẹ Iwe-aṣẹ Ibi ipamọ data Open (ODbL), wo https://world.openfoodfacts.org/terms-of-use fun awọn alaye ofin. Àwọn ipò méjèèjì ni ìfàmọ́ra àti ìpín-bákan náà. Tí o bá so dátà láti Open Food Facts pọ̀ mọ́ àwọn dátà ìpamọ́ mìíràn, nígbà náà ODbL nílò pé kí a tú dátà ìpamọ́ tí ó jáde jáde gẹ́gẹ́ bí dátà tí ó ṣí sílẹ̀ pẹ̀lú. Ó tún túmọ̀ sí wípé o lè so dátà pọ̀ mọ́ àwọn orísun tí yóò gba irú àtúnpín bẹ́ẹ̀ láàyè.

- O gbọdọ lo aṣoju olumulo aṣa nigbagbogbo nigbati o ba n ṣe awọn ipe API lati ṣe idanimọ ohun elo rẹ.

- A fi opin si oṣuwọn fun opin API kọọkan.
