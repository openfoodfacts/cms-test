Lọ́wọ́lọ́wọ́, a kò ní àwọn ohun èlò ìforúkọsílẹ̀ ìtàn (JSONL, MongoDB, CSV).

Sibẹsibẹ, fun awọn ọja kọọkan, o ṣee ṣe lati wọle si awọn ẹya ti tẹlẹ ti data ọja nipa lilo API tabi lori oju-iwe ọja nipa lilo awọn atunṣe.

Nígbàkúgbà tí a bá ṣe àtúnṣe ọjà kan, àtúnṣe tuntun kan (tí ó ń pọ̀ sí i láti 1) ni a máa ń ṣẹ̀dá.

Fún àpẹẹrẹ, láti gba àtúnṣe àkọ́kọ́ (=àtúnṣe ọjà àkọ́kọ́) ti ọjà yìí, lo

https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1.

Bákan náà, a lè lo àtúnṣe rev pẹ̀lú API:

https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1
