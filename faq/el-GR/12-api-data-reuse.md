## Υπάρχουν συστάσεις κάπου στην τεκμηρίωση σχετικά με το ποιο θα ήταν ένα καλό μέγεθος για τις φωτογραφίες που έχουν ανεβάσει;

Αυτό μπορεί να εξαρτάται από τις χώρες, εάν το δίκτυο είναι αργό ή ακριβό. Οτιδήποτε έχει βάρος ή ύψος πάνω από 5000 pixel πιθανότατα δεν είναι πολύ χρήσιμο. και αν μπορείτε με κάποιο τρόπο να εντοπίσετε ότι το δίκτυο είναι αργό, τότε ακόμη και μια εικόνα 2000 pixel θα ήταν εξαιρετική (σίγουρα καλύτερη από το να μην έχετε εικόνα!)

## Τι γίνεται με τα τρόφιμα χωρίς γραμμωτούς κώδικες;

Το Open Food Facts περιέχει μόνο πληροφορίες σχετικά με συσκευασμένα τρόφιμα. Για τις μέσες τιμές των προϊόντων (για παράδειγμα, ντομάτες ή μπανάνες) και άλλων τροφίμων, μπορείτε να χρησιμοποιήσετε μία από τις επίσημες εθνικές βάσεις δεδομένων διατροφής.

**Σημείωση:** Η παρακάτω λίστα περιέχει μερικές από τις πιο σημαντικές εθνικές βάσεις δεδομένων τροφίμων. Αν πιστεύετε ότι κάποια άλλη βάση δεδομένων θα έπρεπε να συμπεριληφθεί στη λίστα, επικοινωνήστε μαζί μας στη διεύθυνση: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## Μπορώ να αναζητήσω ένα ακριβές όνομα προϊόντος με το API;

Δυστυχώς, δεν είναι ακόμη δυνατή η εύκολη αναζήτηση μόνο με βάση το όνομα του προϊόντος και με ακρίβεια μέσω του API.

Η χρήση ενός φίλτρου ανά κατηγορία μπορεί να σας βοηθήσει να κάνετε την αναζήτησή σας πιο ακριβή.

## Πώς μπορώ να έχω πρόσβαση/συλλογή δεδομένων για τα έργα μου;

Στην κύρια σελίδα του Open Food Facts, στην επάνω αριστερή γωνία της οθόνης, υπάρχει ένα μενού κύλισης. Στο κάτω μέρος, θα βρείτε την επιλογή «σύνθετη αναζήτηση», στην οποία μπορείτε να κάνετε κλικ. Στη συνέχεια, εναπόκειται σε εσάς να προσδιορίσετε ποια κριτήρια είναι τα πιο σχετικά με το/τα έργο/α σας. Μόλις επιλέξετε, θα μπορείτε να κατεβάσετε τα αποτελέσματα που έχετε λάβει κάνοντας κύλιση προς τα κάτω στο κάτω μέρος της σελίδας και κάνοντας κλικ στην επιλογή «Λήψη αποτελεσμάτων».

Μπορείτε επίσης να συμβουλευτείτε:

- Η τεκμηρίωση του API μας:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Όροι Χρήσης του Open Food Facts:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Στα δεδομένα μας:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## Υπάρχουν προϋποθέσεις για τη χρήση του API;

Όλη η τεκμηρίωση σχετικά με τη χρήση του API μπορεί να βρεθεί στη [σελίδα τεκμηρίωσης API](https://openfoodfacts.github.io/openfoodfacts-server/api/), αλλά ακολουθεί μια σύντομη περίληψη:

- Η βάση δεδομένων Open Food Facts είναι διαθέσιμη ως ανοιχτά δεδομένα βάσει της Άδειας Ανοικτής Βάσης Δεδομένων (ODbL), ανατρέξτε στη διεύθυνση [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) για τις νομικές λεπτομέρειες. Οι δύο προϋποθέσεις είναι η απόδοση και η ομοιομορφία στην κοινή χρήση. Εάν συνδυάσετε δεδομένα από το Open Food Facts με άλλες βάσεις δεδομένων, τότε ο νόμος ODbL απαιτεί η βάση δεδομένων που προκύπτει να δημοσιευτεί και ως ανοιχτά δεδομένα. Σημαίνει επίσης ότι μπορείτε να συνδυάσετε τα δεδομένα μόνο με πηγές που θα επέτρεπαν τέτοια αναδιανομή.

- Πρέπει **πάντα** να χρησιμοποιείτε έναν προσαρμοσμένο παράγοντα χρήστη (User-Agent) όταν εκτελείτε κλήσεις API για την αναγνώριση της εφαρμογής σας.

- Επιβάλλονται όρια ρυθμού για κάθε τελικό σημείο API.

## Πώς μπορώ να έχω πρόσβαση σε ιστορικά δεδομένα;

Προς το παρόν, δεν προσφέρουμε αρχεία ιστορικών δεδομένων (JSONL, MongoDB, CSV).

Ωστόσο, για μεμονωμένα προϊόντα, είναι δυνατή η πρόσβαση σε προηγούμενες εκδόσεις των δεδομένων προϊόντος χρησιμοποιώντας το API ή στη σελίδα προϊόντος χρησιμοποιώντας αναθεωρήσεις.

Κάθε φορά που ενημερώνεται ένα προϊόν, δημιουργείται μια νέα αναθεώρηση (αυξανόμενο ψηφίο ξεκινώντας από το 1).

Για παράδειγμα, για να λάβετε την πρώτη αναθεώρηση (=πρώτη έκδοση προϊόντος) αυτού του προϊόντος, χρησιμοποιήστε

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Ομοίως, η παράμετρος rev μπορεί να χρησιμοποιηθεί με το API:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
