Όταν βρίσκεστε στην αρχική σελίδα της Πλατφόρμας Παραγωγών:

1. κάντε κλικ στον σύνδεσμο "→ Αριθμός προϊόντων με ευκαιρίες βελτίωσης" στο επάνω μέρος της σελίδας.

2. μπορείτε να συμβουλευτείτε όλες τις ευκαιρίες που εντοπίζει το σύστημά μας για τη βελτίωση των προϊόντων σας

3. Κάνοντας κλικ σε έναν τύπο ευκαιρίας, θα μεταφερθείτε στη λίστα με τα προϊόντα που υπόκεινται σε βελτίωση.

→ Για περισσότερες λεπτομέρειες, σας προσκαλούμε να κατεβάσετε τον οδηγό χρήσης μας ή/και να συμβουλευτείτε τα εκπαιδευτικά μας βίντεο :-)
