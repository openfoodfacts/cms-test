Υπάρχουν μερικοί πόροι που θα σας βοηθήσουν να αξιοποιήσετε στο έπακρο την πλατφόρμα Pro.

Είστε ευπρόσδεκτοι να:

- Ελέγξτε τον οδηγό χρήσης μας - σας προσκαλούμε να κατεβάσετε το PDF, ώστε να μπορείτε να κάνετε κλικ στους συνδέσμους, κάτι που θα σας διευκολύνει στην πλοήγησή σας στο έγγραφο.

- συμβουλευτείτε τα εκπαιδευτικά μας βίντεο

- μοιραστείτε σχόλια ή προτάσεις μέσω αυτής της φόρμας

- Κλείστε ένα τηλεφώνημα με τη Manon, τη διευθύντρια συνεργασιών μας, εάν έχετε περισσότερες ερωτήσεις: εδώ

- Επικοινωνήστε μαζί μας μέσω email: producers@openfoodfacts.org
