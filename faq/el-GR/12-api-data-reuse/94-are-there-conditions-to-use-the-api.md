Όλη η τεκμηρίωση σχετικά με τη χρήση του API μπορεί να βρεθεί στη σελίδα τεκμηρίωσης του API, αλλά ακολουθεί μια σύντομη περίληψη:

- Η βάση δεδομένων Open Food Facts είναι διαθέσιμη ως ανοιχτά δεδομένα βάσει της Άδειας Ανοικτής Βάσης Δεδομένων (ODbL), ανατρέξτε στη διεύθυνση https://world.openfoodfacts.org/terms-of-use για τις νομικές λεπτομέρειες. Οι δύο προϋποθέσεις είναι η απόδοση και η ομοιομορφία στην κοινή χρήση. Εάν συνδυάσετε δεδομένα από το Open Food Facts με άλλες βάσεις δεδομένων, τότε ο νόμος ODbL απαιτεί η βάση δεδομένων που προκύπτει να δημοσιευτεί και ως ανοιχτά δεδομένα. Σημαίνει επίσης ότι μπορείτε να συνδυάσετε τα δεδομένα μόνο με πηγές που θα επέτρεπαν τέτοια αναδιανομή.

- Πρέπει πάντα να χρησιμοποιείτε έναν προσαρμοσμένο παράγοντα χρήστη (User-Agent) όταν εκτελείτε κλήσεις API για την αναγνώριση της εφαρμογής σας.

- Επιβάλλονται όρια ρυθμού για κάθε τελικό σημείο API.
