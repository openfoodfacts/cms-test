## ھۆججەتلەردە يۈكلەنگەن سۈرەتلەرنىڭ قايسى چوڭلۇقتا بولۇشى كېرەكلىكى توغرىسىدا تەۋسىيە بارمۇ؟

بۇ تور سۈرئىتىنىڭ ئاستا ياكى قىممەت بولۇشى دۆلەتلەرگە باغلىق بولۇشى مۇمكىن. ئېغىرلىقى ياكى بوي ئېگىزلىكى 5000 پىكسېلدىن ئېشىپ كەتكەن ھەر قانداق نەرسە ئانچە پايدىلىق بولماسلىقى مۇمكىن. ئەگەر تورنىڭ سۈرئىتىنىڭ ئاستا ئىكەنلىكىنى قانداقلا بولمىسۇن بايقىسىڭىز، 2000 پىكسېللىق رەسىممۇ ناھايىتى ياخشى بولاتتى (رەسىمسىز بولغاندىن كۆرە ئەلۋەتتە ياخشى!)

## شىتىرىكودسىز يېمەكلىكلەرچۇ؟

«ئوچۇق يېمەكلىك پاكىتلىرى» پەقەت ئورالغان يېمەكلىكلەر ھەققىدىكى ئۇچۇرلارنىلا ئۆز ئىچىگە ئالىدۇ. مەھسۇلاتلارنىڭ (مەسىلەن، پەمىدۇر ياكى بانان) ۋە باشقا يېمەكلىكلەرنىڭ ئوتتۇرىچە قىممىتى ئۈچۈن، سىز رەسمىي دۆلەتلىك ئوزۇقلۇق سانلىق مەلۇمات ئامبىرىنىڭ بىرىنى ئىشلەتسىڭىز بولىدۇ.

**ئەسكەرتىش:** تۆۋەندىكى تىزىملىكتە ئەڭ مۇھىم دۆلەتلىك يېمەكلىك سانلىق مەلۇمات ئامبىرىنىڭ بىر قىسمى بار. ئەگەر سىز باشقا سانلىق مەلۇمات ئامبىرىنى تىزىملىككە كىرگۈزۈش كېرەك دەپ ئويلىسىڭىز، بىز بىلەن ئالاقىلىشىڭ: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

**مىللىي يېمەكلىك سانلىق مەلۇمات ئامبىرى تىزىملىكى**

## API ئارقىلىق ئېنىق مەھسۇلات نامىنى ئىزدىيەلەمدىم؟

ئەپسۇس، ھازىرچە پەقەت مەھسۇلات نامى ئارقىلىق ۋە API ئارقىلىق ئاسانلا ئىزدەش مۇمكىن ئەمەس.

تۈرگە سۈزگۈچ ئىشلىتىش ئىزدەشنى تېخىمۇ ئېنىق قىلىشقا ياردەم بېرىشى مۇمكىن.

---

## تۈرلىرىم ئۈچۈن سانلىق مەلۇماتلارغا قانداق ئېرىشەلەيمەن/توپلىيالايمەن؟

«ئوچۇق يېمەكلىك ئۇچۇرلىرى» نىڭ ئاساسلىق بېتىدە، ئېكراننىڭ ئۈستۈنكى سول بۇلۇڭىدا، سىيرىلما تىزىملىك بار. ئۇنىڭ ئاستى تەرىپىدە، سىز چېكىش ئارقىلىق «ئىلغار ئىزدەش» تاللانمىسىنى تاپالايسىز. ئۇنداقتا قايسى ئۆلچەملەرنىڭ تۈرىڭىزگە ئەڭ ماس كېلىدىغانلىقىنى بەلگىلەش سىزگە باغلىق. تاللىغاندىن كېيىن، بەتنىڭ ئاستى تەرىپىگە قاراپ سىيرىلىپ، «نەتىجىلەرنى چۈشۈرۈش» نى چېكىش ئارقىلىق ئېرىشكەن نەتىجىلەرنى چۈشۈرەلەيسىز.

سىز يەنە تۆۋەندىكىلەر بىلەن مەسلىھەتلىشىپ كۆرسىڭىز بولىدۇ:

- بىزنىڭ API ھۆججەتلىرىمىز:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- ئوچۇق يېمەكلىك پاكىتلىرىنىڭ ئىشلىتىش شەرتلىرى:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- بىزنىڭ سانلىق مەلۇماتلىرىمىزدا:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## API نى ئىشلىتىش ئۈچۈن شەرتلەر بارمۇ؟

API ئىشلىتىش توغرىسىدىكى بارلىق ھۆججەتلەرنى [API ھۆججەتلىرى بېتى](https://openfoodfacts.github.io/openfoodfacts-server/api/) دىن تاپقىلى بولىدۇ، ئەمما بۇ يەردە قىسقىچە خۇلاسە بار:

- ئوچۇق يېمەكلىك ئۇچۇرلىرى سانلىق مەلۇمات ئامبىرى ئوچۇق سانلىق مەلۇمات ئامبىرى ئىجازەتنامىسى (ODbL) بويىچە ئوچۇق سانلىق مەلۇمات سۈپىتىدە تەمىنلىنىدۇ، قانۇنىي تەپسىلاتلار ئۈچۈن [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) غا قاراڭ. بۇ ئىككى شەرت ئىگىدارلىق ھوقۇقى ۋە ئورتاقلىشىشتىن ئىبارەت. ئەگەر سىز ئوچۇق يېمەكلىك پاكىتلىرىدىكى سانلىق مەلۇماتلارنى باشقا سانلىق مەلۇماتلار ئامبىرى بىلەن بىرلەشتۈرسىڭىز، ODbL نەتىجىدە ھاسىل بولغان سانلىق مەلۇمات ئامبىرىنىڭ ئوچۇق سانلىق مەلۇمات سۈپىتىدە تارقىتىلىشىنى تەلەپ قىلىدۇ. بۇ يەنە سانلىق مەلۇماتلارنى پەقەت مۇشۇنداق قايتا تەقسىملەشكە يول قويىدىغان مەنبەلەر بىلەن بىرلەشتۈرگىلى بولىدىغانلىقىدىن دېرەك بېرىدۇ.

- ئەپنى ئېنىقلاش ئۈچۈن API چاقىرىشلىرىنى ئىجرا قىلغاندا، سىز ھەمىشە خاسلاشتۇرۇلغان ئىشلەتكۈچى ۋاكالەتچىسىنى ئىشلىتىشىڭىز كېرەك.

- ھەر بىر API ئاخىرقى نۇقتىسى ئۈچۈن نىسبەت چەكلىمىسى يولغا قويۇلىدۇ.

## تارىخىي سانلىق مەلۇماتلارغا قانداق ئېرىشەلەيمەن؟

ھازىر بىز تارىخىي سانلىق مەلۇماتلارنى (JSONL، MongoDB، CSV) ساقلىمايمىز.

قانداقلا بولمىسۇن، يەككە مەھسۇلاتلار ئۈچۈن، مەھسۇلات سانلىق مەلۇماتلىرىنىڭ ئالدىنقى نەشرىگە API ئارقىلىق ياكى مەھسۇلات بېتىدە تۈزىتىشلەر ئارقىلىق كىرگىلى بولىدۇ.

مەھسۇلات ھەر قېتىم يېڭىلانغاندا، يېڭى نەشرى (1 دىن باشلاپ چوڭىيىدىغان رەقەم) ھاسىل قىلىنىدۇ.

مەسىلەن، بۇ مەھسۇلاتنىڭ تۇنجى نەشرىنى (=بىرىنچى مەھسۇلات نۇسخىسى) ئېلىش ئۈچۈن، ئىشلىتىڭ

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

شۇنىڭغا ئوخشاش، rev پارامېتىرىنى API بىلەن بىللە ئىشلىتىشكە بولىدۇ:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
