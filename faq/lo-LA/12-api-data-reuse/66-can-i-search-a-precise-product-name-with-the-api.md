ແຕ່ຫນ້າເສຍດາຍທີ່ມັນຍັງບໍ່ສາມາດຄົ້ນຫາໄດ້ງ່າຍດ້ວຍຊື່ຜະລິດຕະພັນເທົ່ານັ້ນ ແລະ ຄົ້ນຫາຜ່ານ API ໄດ້ຢ່າງແນ່ນອນ.

ການໃຊ້ຕົວກອງໃນໝວດໝູ່ອາດຊ່ວຍໃຫ້ການຄົ້ນຫາຂອງທ່ານມີຄວາມແນ່ນອນຫຼາຍຂຶ້ນໄດ້.
