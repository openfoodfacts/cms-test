Open Food Facts API ແມ່ນຄືກັນກັບ Open Beauty Facts API 98% ສະນັ້ນຖ້າທ່ານໃຊ້ເອກະສານ Open Food Facts API ທ່ານຄວນຈະມີມຸມມອງທີ່ຊັດເຈນກ່ຽວກັບ Open Beauty Facts API.

ໃຫ້ສັງເກດວ່າຄຸນສົມບັດຕ່າງໆເຊັ່ນ: ແຜງຄວາມຮູ້, Eco-Score, Nutri-Score ແມ່ນບໍ່ຮອງຮັບ (ບໍ່ວ່າຈະເປັນຍ້ອນຫົວຂໍ້ ຫຼື ຍ້ອນຖານລະຫັດເກົ່າ).
