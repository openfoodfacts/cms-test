API foydalanish haqidagi barcha hujjatlarni API hujjatlari sahifasida topish mumkin, ammo bu yerda qisqacha xulosa:

- Ochiq oziq-ovqat ma'lumotlari ma'lumotlar bazasi Ochiq ma'lumotlar bazasi litsenziyasi (ODbL) bo'yicha ochiq ma'lumotlar sifatida mavjud, huquqiy tafsilotlar uchun https://world.openfoodfacts.org/terms-of-use sahifasiga qarang. Ikki shart - atribut va teng ulush. Agar siz Open Food Facts ma'lumotlarini boshqa ma'lumotlar bazalari bilan birlashtirsangiz, ODbL natijada olingan ma'lumotlar bazasini ham ochiq ma'lumotlar sifatida chiqarishni talab qiladi. Bu shuningdek, ma'lumotlarni faqat bunday qayta taqsimlashga imkon beradigan manbalar bilan birlashtirishingiz mumkinligini anglatadi.

- Ilovangizni aniqlash uchun API qo'ng'iroqlarini amalga oshirishda har doim maxsus User-Agentdan foydalanishingiz kerak.

- Har bir API so'nggi nuqtasi uchun tezlik chegaralari qo'llaniladi.
