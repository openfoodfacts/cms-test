## Yuklangan fotosuratlar uchun qaysi o'lcham yaxshi bo'lishi haqida hujjatlarda biron bir tavsiya bormi?

Bu tarmoq sekin yoki qimmat bo'lsa, mamlakatlarga bog'liq bo'lishi mumkin. Og'irligi yoki bo'yi 5000 pikseldan yuqori bo'lgan har qanday narsa, ehtimol, juda foydali emas. va agar siz tarmoqning sekin ishlashini qandaydir yo'l bilan aniqlasangiz, unda hatto 2000 pikselli tasvir ham ajoyib bo'lar edi (albatta, tasvirga ega bo'lmaslikdan ko'ra yaxshiroq!)

## Shtrixkodsiz ovqat haqida nima deyish mumkin?

Ochiq oziq-ovqat faktlari faqat qadoqlangan oziq-ovqat haqida ma'lumotlarni o'z ichiga oladi. Mahsulotlar (masalan, pomidor yoki banan) va boshqa oziq-ovqat mahsulotlarining o'rtacha qiymatlari uchun siz rasmiy milliy ovqatlanish ma'lumotlar bazalaridan birini ishlatishingiz mumkin.

**Izoh:** Quyidagi ro'yxatda eng muhim milliy oziq-ovqat ma'lumotlar bazalari keltirilgan. Agar ro'yxatga boshqa ma'lumotlar bazasini kiritish kerak deb hisoblasangiz, iltimos, biz bilan quyidagi manzil orqali bog'laning: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## API yordamida aniq mahsulot nomini qidira olamanmi?

Afsuski, faqat mahsulot nomi bo'yicha va aniq API orqali osongina qidirish hali mumkin emas.

Kategoriya bo'yicha filtrdan foydalanish qidiruvingizni aniqroq qilishga yordam berishi mumkin.

## Loyihalarim uchun ma'lumotlarga qanday kirishim/to'plashim mumkin?

Ochiq oziq-ovqat ma'lumotlari bosh sahifasida, ekranning yuqori chap burchagida aylantirish menyusi mavjud. Uning pastki qismida siz bosishingiz mumkin bo'lgan "kengaytirilgan qidiruv" opsiyasini topasiz. Keyin qaysi mezonlar sizning loyihangiz(lar) uchun eng mos kelishini aniqlash sizga bog'liq. Tanlangandan so'ng, sahifaning pastki qismida pastga aylantirib va "Natijalarni yuklab olish" tugmasini bosish orqali olingan natijalarni yuklab olishingiz mumkin bo'ladi.

Shuningdek, quyidagilarga murojaat qilishingiz mumkin:

- Bizning API hujjatlarimiz:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Open Food Facts’ Foydalanish shartlari:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Ma'lumotlarimiz bo'yicha:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## APIdan foydalanish uchun shartlar bormi?

API foydalanish haqidagi barcha hujjatlarni [API hujjatlari sahifasida](https://openfoodfacts.github.io/openfoodfacts-server/api/) topish mumkin, ammo bu yerda qisqacha xulosa:

- Ochiq oziq-ovqat ma'lumotlari ma'lumotlar bazasi Ochiq ma'lumotlar bazasi litsenziyasi (ODbL) bo'yicha ochiq ma'lumotlar sifatida mavjud, huquqiy tafsilotlar uchun [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) ga qarang. Ikki shart - atribut va teng ulush. Agar siz Open Food Facts ma'lumotlarini boshqa ma'lumotlar bazalari bilan birlashtirsangiz, ODbL natijada olingan ma'lumotlar bazasini ham ochiq ma'lumotlar sifatida chiqarishni talab qiladi. Bu shuningdek, ma'lumotlarni faqat bunday qayta taqsimlashga imkon beradigan manbalar bilan birlashtirishingiz mumkinligini anglatadi.

- Ilovangizni aniqlash uchun API chaqiruvlarini amalga oshirishda siz har doim maxsus User-Agentdan foydalanishingiz kerak.

- Har bir API so'nggi nuqtasi uchun tezlik chegaralari qo'llaniladi.

## Tarixiy ma'lumotlarga qanday kirishim mumkin?

Hozirda biz tarixiy ma'lumotlarni (JSONL, MongoDB, CSV) saqlashni taklif qilmaymiz.

Biroq, individual mahsulotlar uchun mahsulot ma'lumotlarining oldingi versiyalariga API yordamida yoki mahsulot sahifasida tuzatishlar yordamida kirish mumkin.

Har safar mahsulot yangilanganda, yangi tahrir (1 dan boshlab ortib boruvchi raqam) yaratiladi.

Masalan, ushbu mahsulotning birinchi versiyasini (=birinchi mahsulot versiyasini) olish uchun foydalaning

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Xuddi shunday, rev parametri API bilan ishlatilishi mumkin:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
