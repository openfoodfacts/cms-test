Open Food Facts API je na 98 % rovnaké ako Open Beauty Facts API, takže ak používate dokumentáciu k Open Food Facts API, mali by ste mať o Open Beauty Facts API celkom jasný prehľad.

Upozorňujeme, že funkcie ako Panely znalostí, Eco-Score a Nutri-Score nie sú podporované (buď kvôli téme, alebo kvôli starej kódovej základni).
