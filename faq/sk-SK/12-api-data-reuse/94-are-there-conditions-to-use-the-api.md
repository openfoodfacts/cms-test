Všetku dokumentáciu o používaní API nájdete na stránke s dokumentáciou k API, ale tu je stručný súhrn:

- Databáza Open Food Facts je dostupná ako otvorené dáta pod licenciou Open Database License (ODbL), právne podrobnosti nájdete na stránke https://world.openfoodfacts.org/terms-of-use. Tieto dve podmienky sú priradenie a zdieľanie za rovnakých podmienok. Ak kombinujete údaje z Open Food Facts s inými databázami, ODbL vyžaduje, aby výsledná databáza bola tiež zverejnená ako otvorené údaje. To tiež znamená, že dáta môžete kombinovať iba so zdrojmi, ktoré by takúto redistribúciu umožnili.

- Pri vykonávaní volaní API musíte vždy použiť vlastného používateľského agenta na identifikáciu svojej aplikácie.

- Pre každý koncový bod API sa vynucujú limity rýchlosti.
