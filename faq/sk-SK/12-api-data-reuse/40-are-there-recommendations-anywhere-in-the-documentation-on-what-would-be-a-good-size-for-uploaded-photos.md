To môže závisieť od krajiny, či je sieť pomalá alebo drahá. čokoľvek s hmotnosťou alebo výškou nad 5000 pixelov pravdepodobne nie je veľmi užitočné. a ak nejako dokážete zistiť, že sieť je pomalá, potom by bol skvelý aj obrázok s 2000 pixelmi (určite lepší ako žiadny obrázok!).
