## अपलोड गरिएका तस्बिरहरूको लागि कुन आकार राम्रो हुन्छ भन्ने बारेमा कागजातमा कतै सिफारिसहरू छन्?

नेटवर्क ढिलो छ वा महँगो छ भन्ने कुरा देशहरूमा निर्भर हुन सक्छ। ५००० पिक्सेलभन्दा माथिको तौल वा उचाइ सायद धेरै उपयोगी हुँदैन। र यदि तपाईंले कुनै तरिकाले नेटवर्क ढिलो छ भनेर पत्ता लगाउन सक्नुहुन्छ भने, २००० पिक्सेलको तस्बिर पनि राम्रो हुनेछ (छवि नहुनु भन्दा निश्चित रूपमा राम्रो!)

## बारकोड बिनाको खानाको बारेमा के हुन्छ?

ओपन फुड फ्याक्ट्समा प्याकेज गरिएको खानाको बारेमा मात्र जानकारी हुन्छ। उत्पादनको औसत मूल्य (उदाहरणका लागि, गोलभेडा वा केरा) र अन्य खाद्य उत्पादनहरूको लागि, तपाईंले आधिकारिक राष्ट्रिय पोषण डाटाबेसहरू मध्ये एक प्रयोग गर्न सक्नुहुन्छ।

**नोट:** तलको सूचीमा केही महत्त्वपूर्ण राष्ट्रिय खाद्य डाटाबेसहरू छन्। यदि तपाईंलाई लाग्छ कि सूचीमा अन्य कुनै डाटाबेस समावेश गर्नुपर्छ भने, कृपया हामीलाई यहाँ सम्पर्क गर्नुहोस्: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## के म API मार्फत सटीक उत्पादन नाम खोज्न सक्छु?

दुर्भाग्यवश API मार्फत उत्पादनको नाममा मात्र र सटीक रूपमा सजिलै खोज्न सम्भव छैन।

श्रेणीमा फिल्टर प्रयोग गर्नाले तपाईंको खोजलाई अझ सटीक बनाउन मद्दत गर्न सक्छ।

---

## म मेरा परियोजनाहरूको लागि डेटा कसरी पहुँच/संकलन गर्न सक्छु?

ओपन फूड फ्याक्ट्सको मुख्य पृष्ठमा, स्क्रिनको माथिल्लो बायाँ कुनामा, स्क्रोलिङ मेनु छ। यसको तल, तपाईंले "उन्नत खोज" विकल्प पाउनुहुनेछ, जसमा तपाईं क्लिक गर्न सक्नुहुन्छ। त्यसपछि तपाईंको परियोजना(हरू) को लागि कुन मापदण्ड सबैभन्दा सान्दर्भिक छ भनेर निर्धारण गर्ने काम तपाईंमा निर्भर छ। एकपटक छनौट गरिसकेपछि, तपाईंले पृष्ठको तल स्क्रोल गरेर र "परिणामहरू डाउनलोड गर्नुहोस्" मा क्लिक गरेर प्राप्त परिणामहरू डाउनलोड गर्न सक्षम हुनुहुनेछ।

तपाईं पनि परामर्श गर्न सक्नुहुन्छ:

- हाम्रो API कागजात:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- ओपन फुड फ्याक्ट्सको प्रयोगका सर्तहरू:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- हाम्रो डेटामा:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## के API प्रयोग गर्न कुनै सर्तहरू छन्?

API प्रयोगको बारेमा सबै कागजातहरू [API कागजात पृष्ठ](https://openfoodfacts.github.io/openfoodfacts-server/api/) मा फेला पार्न सकिन्छ, तर यहाँ एउटा छोटो सारांश छ:

- ओपन फुड फ्याक्ट्स डाटाबेस ओपन डाटाबेस लाइसेन्स (ODbL) अन्तर्गत खुला डाटाको रूपमा उपलब्ध छ, कानुनी विवरणहरूको लागि [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) हेर्नुहोस्। दुई सर्तहरू एट्रिब्युशन र सेयर-एलाइक हुन्। यदि तपाईंले ओपन फूड फ्याक्ट्सबाट प्राप्त डेटालाई अन्य डाटाबेसहरूसँग संयोजन गर्नुभयो भने, ODbL ले परिणामस्वरूप डाटाबेसलाई पनि खुला डेटाको रूपमा जारी गर्न आवश्यक छ। यसको अर्थ यो पनि हो कि तपाईंले त्यस्तो पुनर्वितरणलाई अनुमति दिने स्रोतहरूसँग मात्र डेटा संयोजन गर्न सक्नुहुन्छ।

- आफ्नो एप पहिचान गर्न API कल गर्दा तपाईंले **सधैं** अनुकूलन प्रयोगकर्ता-एजेन्ट प्रयोग गर्नुपर्छ।

- प्रत्येक API अन्त्यबिन्दुको लागि दर-सीमाहरू लागू गरिन्छ।

## म कसरी ऐतिहासिक डेटा पहुँच गर्न सक्छु?

हाल, हामी ऐतिहासिक डेटा डम्प (JSONL, MongoDB, CSV) प्रस्ताव गर्दैनौं।

यद्यपि, व्यक्तिगत उत्पादनहरूको लागि, API प्रयोग गरेर वा उत्पादन पृष्ठमा संशोधनहरू प्रयोग गरेर उत्पादन डेटाको अघिल्लो संस्करणहरू पहुँच गर्न सम्भव छ।

प्रत्येक पटक उत्पादन अद्यावधिक गर्दा, नयाँ संशोधन (१ बाट सुरु हुने बढ्दो अंक) सिर्जना गरिन्छ।

उदाहरणका लागि, यस उत्पादनको पहिलो संशोधन (=पहिलो उत्पादन संस्करण) प्राप्त गर्न, प्रयोग गर्नुहोस्

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).
