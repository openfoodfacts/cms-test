দুর্ভাগ্যবশত API-এর মাধ্যমে কেবল পণ্যের নামে এবং সঠিকভাবে অনুসন্ধান করা এখনও সম্ভব হয়নি।

তবে ক্যাটাগরি অনুযায়ী ফিল্টার ব্যবহার করলে আপনার অনুসন্ধান আরও সুনির্দিষ্ট হতে পারে।
