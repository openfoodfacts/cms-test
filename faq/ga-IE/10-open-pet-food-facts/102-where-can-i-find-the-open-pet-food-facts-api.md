Tá API Fíricí Bia Oscailte Pet 98% mar an gcéanna le API Fíricí Bia Oscailte, mar sin má úsáideann tú doiciméadacht API Fíricí Bia Oscailte Peataí ba cheart go mbeadh léargas soiléir agat ar API Fíricí Bia Oscailte Pet.

Tabhair faoi deara nach dtacaítear le gnéithe cosúil leis na Painéil Eolais, an Scór Éicea, agus an Scór Cothaithe (mar gheall ar an topaic, nó mar gheall ar an seanbhunachar cód).
