Is féidir an doiciméadú go léir faoi úsáid API a fháil ar leathanach doiciméadú API, ach seo achoimre ghearr:

- Tá bunachar sonraí Open Food Facts ar fáil mar shonraí oscailte faoin gCeadúnas Bunachar Sonraí Oscailte (ODbL), féach https://world.openfoodfacts.org/terms-of-use le haghaidh na sonraí dlíthiúla. Is iad an dá choinníoll ná leithdháileadh agus roinnt chomhionann. Má chomhcheanglaíonn tú sonraí ó Open Food Facts le bunachair shonraí eile, éilíonn ODbL go gcaithfear an bunachar sonraí a eascraíonn as sin a scaoileadh mar shonraí oscailte chomh maith. Ciallaíonn sé freisin nach féidir leat na sonraí a chomhcheangal ach le foinsí a cheadódh athdháileadh den sórt sin.

- Ní mór duit Gníomhaire Úsáideora saincheaptha a úsáid i gcónaí agus glaonna API á ndéanamh agat chun d'aip a aithint.

- Cuirtear teorainneacha ráta i bhfeidhm do gach críochphointe API.
