API Open Pet Food Facts giống API Open Food Facts đến 98%, vì vậy nếu bạn sử dụng tài liệu của API Open Food Facts, bạn sẽ có cái nhìn khá rõ ràng về API Open Pet Food Facts.

Xin lưu ý rằng các tính năng như Bảng kiến thức, Điểm sinh thái, Điểm dinh dưỡng hiện không được hỗ trợ (có thể do chủ đề hoặc do mã nguồn cũ).
