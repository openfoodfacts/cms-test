Sólo los moderadores pueden eliminar fotos, para evitar posibles actos vandálicos.

Sólo tienes que pedir en Slack o en contact@openfoodfacts.org para eliminar los duplicados o cualquier fotos inapropiadas (usted debe tratar de proporcionar el número de código de barras o URL para así).

También tenemos una nueva API de informe de imágenes si eres programador.
