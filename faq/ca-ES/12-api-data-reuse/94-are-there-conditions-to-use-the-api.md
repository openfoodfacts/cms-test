Tota la documentació sobre l'ús de l'API es pot trobar a la pàgina de documentació de l'API, però aquí teniu un resum ràpid:

- La base de dades Open Food Facts està disponible com a dades obertes sota la Llicència de Base de Dades Oberta (ODbL); vegeu https://world.openfoodfacts.org/terms-of-use per obtenir més informació legal. Les dues condicions són atribució i compartició igual. Si combineu dades d'Open Food Facts amb altres bases de dades, l'ODbL exigeix que la base de dades resultant també es publiqui com a dades obertes. També significa que només podeu combinar les dades amb fonts que permetin aquesta redistribució.

- Sempre has d'utilitzar un agent d'usuari personalitzat quan realitzis crides a l'API per identificar la teva aplicació.

- Els límits de velocitat s'apliquen per a cada punt final de l'API.
