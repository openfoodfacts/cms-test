Open Food Facts només conté informació sobre aliments envasats. Per als valors mitjans dels productes agrícoles (per exemple, tomàquets o plàtans) i altres productes alimentaris, podeu utilitzar una de les bases de dades nacionals oficials de nutrició.

Nota: La llista següent conté algunes de les bases de dades nacionals d'aliments més importants. Si creieu que s'hauria d'incloure alguna altra base de dades a la llista, poseu-vos en contacte amb nosaltres a: https://world.openfoodfacts.org/contact

Llista de bases de dades nacionals d'aliments

- Austràlia - FSANZ - NUTTAB 2006: https://www.foodstandards.gov.au/media/documents/FSANZ%20Conf%20PostersNUTTAB.pdf
- Bèlgica - NUBEL - Dades belgues sobre la composició d'aliments: https://www.internubel.be
- Canadà - FCEN: https://aliments-nutrition.canada.ca/cnf-fce/index-fra.jsp
- República Txeca - Base de dades de composició d'aliments a l'Institut Nacional de Salut Pública: http://www.chpr.szu.cz/dbdata/foodcomp/nut2001.asp
- Dinamarca - Banc de dades de composició d'aliments danès: https://frida.fooddata.dk/?lang=en
- Estònia - Base de dades estoniana de composició d'aliments: https://tka.nutridata.ee/en/
- Finlàndia - Base de dades finlandesa de composició d'aliments - FINELI: https://fineli.fi/fineli/en/index
- França - CIQUAL: https://www.anses.fr/en/search/site/Table%20ciqual
- Alemanya - Base de dades en línia Souci-Fachmann-Kraut: https://www.sfk.online/#/home o la base de dades oficial alemanya: Bundeslebensmittelschlüssel: https://blsdb.de/
- Itàlia - Banca Dati di Composizione degli Alimenti CREA: https://www.crea.gov.it/web/alimenti-e-nutrizione/banche-dati
- Països Baixos - Base de dades neerlandesa de composició d'aliments: https://www.rivm.nl/en/dutch-food-composition-database
- Noruega - Taula de composició d'aliments noruega 2006: https://www.matvaretabellen.no/?language=en
- Polònia - Taules de composició d'aliments: http://www.izz.waw.pl/en/?lang=en
- Espanya - Base de dades espanyola de composició d'aliments - BEDCA: https://www.bedca.net/bdpub/index.php
- Suïssa - Base de dades suïssa de composició d'aliments: https://www.naehrwertdaten.ch/de/
- Regne Unit - Conjunt de dades integrat de composició dels aliments (CoFID): https://www.gov.uk/government/publications/composition-of-foods-integrated-dataset-cofid
- EUA - USDA: https://ndb.nal.usda.gov/
