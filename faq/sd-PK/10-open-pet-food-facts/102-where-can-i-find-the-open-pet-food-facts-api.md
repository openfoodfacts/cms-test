اوپن پيٽ فوڊ فيڪٽس API 98٪ اوپن فوڊ فيڪٽس API جي برابر آهي، تنهن ڪري جيڪڏهن توهان اوپن فوڊ فيڪٽس API دستاويز استعمال ڪندا آهيو ته توهان کي اوپن پيٽ فوڊ فيڪٽس API جو هڪ صاف نظارو هجڻ گهرجي.
