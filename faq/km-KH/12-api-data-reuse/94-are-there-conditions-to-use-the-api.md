ឯកសារទាំងអស់អំពីការប្រើប្រាស់ API អាចរកបាននៅលើទំព័រឯកសារ API ប៉ុន្តែនេះគឺជាសេចក្តីសង្ខេបរហ័ស៖

- មូលដ្ឋានទិន្នន័យ Open Food Facts អាចរកបានជាទិន្នន័យបើកចំហក្រោមអាជ្ញាប័ណ្ណ Open Database (ODbL) សូមមើល https://world.openfoodfacts.org/terms-of-use សម្រាប់ព័ត៌មានលម្អិតផ្នែកច្បាប់។ លក្ខខណ្ឌទាំងពីរគឺ ការបញ្ជាក់កម្មសិទ្ធិ និងការចែករំលែកដូចគ្នា។ ប្រសិនបើអ្នកផ្សំទិន្នន័យពី Open Food Facts ជាមួយមូលដ្ឋានទិន្នន័យផ្សេងទៀត នោះ ODbL តម្រូវឱ្យមូលដ្ឋានទិន្នន័យលទ្ធផលត្រូវតែចេញផ្សាយជាទិន្នន័យបើកចំហផងដែរ។ វាក៏មានន័យថា អ្នកអាចផ្សំទិន្នន័យបានតែជាមួយប្រភពដែលនឹងអនុញ្ញាតឱ្យមានការចែកចាយឡើងវិញបែបនេះ។

- អ្នកត្រូវតែប្រើ User-Agent ផ្ទាល់ខ្លួនជានិច្ច នៅពេលអនុវត្តការហៅ API ដើម្បីកំណត់អត្តសញ្ញាណកម្មវិធីរបស់អ្នក។

- ដែនកំណត់អត្រាត្រូវបានអនុវត្តសម្រាប់ចំណុចបញ្ចប់ API នីមួយៗ។
