Se voi riippua maista, onko verkko hidas tai kallis. Mikään yli 5000 pikselin painoinen tai korkea kohde ei luultavasti ole kovin hyödyllinen. ja jos pystyt jotenkin havaitsemaan, että verkko on hidas, niin jopa 2000 pikselin kuva olisi loistava (varmasti parempi kuin ei kuvaa ollenkaan!).
