所有關於 API 使用方法的文件都可以在 API 文件頁面找到，但在這裡做一個簡要概述：

- Open Food Facts 資料庫根據開放資料庫許可證 (ODbL) 作為開放資料提供，有關法律詳情，請參閱 https://world.openfoodfacts.org/terms-of-use。 這兩個條件是歸屬和相同方式共享。 如果將 Open Food Facts 的資料與其他資料庫的資料結合起來，那麼 ODbL 要求產生的資料庫也必須作為開放資料發布。 這也意味著，你只能將數據與允許此類重新分發的來源結合。

- 在執行 API 呼叫時，必須始終使用自訂 User-Agent 來識別您的應用程式。

- 每個 API 端點都強制執行速率限制。
