Open Pet Food Facts API 與 Open Food Facts API 有 98% 的相似度，因此，如果您使用 Open Food Facts API 文檔，您應該對 Open Pet Food Facts API 有一個相當清晰的了解。

請注意，知識面板、生態評分、營養評分等功能不受支援（可能是由於主題原因，也可能是由於程式碼庫較舊）。

https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/
