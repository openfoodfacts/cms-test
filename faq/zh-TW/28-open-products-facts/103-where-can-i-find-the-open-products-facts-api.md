Open Products Facts API 與 Open Food Facts API 有 98% 的相似度，因此，如果您使用 Open Food Facts API 文件，您應該可以對 Open Products Facts API 有一個相當清晰的了解。 我們還編寫了專門的教程，指導您如何在應用程式中實現通用掃描。
