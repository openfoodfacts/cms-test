„Open Pet Food Facts“ API yra 98 % tokia pati kaip „Open Food Facts“ API, todėl jei naudojate „Open Food Facts“ API dokumentaciją, turėtumėte gana aiškiai suprasti „Open Pet Food Facts“ API.

Atkreipkite dėmesį, kad tokios funkcijos kaip žinių skydeliai, „Eco-Score“, „Nutri-Score“ nepalaikomos (dėl temos arba senos kodų bazės).
