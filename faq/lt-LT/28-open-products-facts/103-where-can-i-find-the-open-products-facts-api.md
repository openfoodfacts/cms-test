„Open Products Facts“ API yra 98 % tokia pati kaip „Open Food Facts“ API, todėl jei naudojate „Open Food Facts“ API dokumentaciją, turėtumėte gana aiškiai suprasti „Open Products Facts“ API. Taip pat parašėme specialų vadovėlį, kuris leis įdiegti universalų nuskaitymą jūsų programėlėje.
