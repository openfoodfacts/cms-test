Visą API naudojimo dokumentaciją galite rasti API dokumentacijos puslapyje, tačiau pateikiame trumpą santrauką:

- „Open Food Facts“ duomenų bazė prieinama kaip atvirieji duomenys pagal Atvirosios duomenų bazės licenciją (ODbL), teisinę informaciją rasite adresu https://world.openfoodfacts.org/terms-of-use. Dvi sąlygos yra priskyrimas ir bendrinimas vienodu principu. Jei jungiate duomenis iš „Open Food Facts“ su kitomis duomenų bazėmis, ODbL reikalauja, kad gauta duomenų bazė taip pat būtų paskelbta kaip atviri duomenys. Tai taip pat reiškia, kad duomenis galite derinti tik su tais šaltiniais, kurie leistų tokį perskirstymą.

- Atlikdami API iškvietimus programai identifikuoti, visada turite naudoti pasirinktinį vartotojo agentą.

- Kiekvienam API galiniam taškui taikomi greičio apribojimai.
