Pour les produits sans code barre, sur le site ou sur la plateforme producteurs, il y a un bouton "Produit sans code barre" dans la colonne de gauche qui permet de les ajouter. Un identifiant est alors généré automatiquement.

Si vous passez par le tableur Excel pour importer vos produits, il convient de reporter cet identifiant dans la colonne "code-barre" pour que le système alimente la fiche produit correctement.
