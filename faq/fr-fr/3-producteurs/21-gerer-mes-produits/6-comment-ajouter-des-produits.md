Pour ajouter des produits sur notre plateforme rien de plus simple :

- Ajoutez un document type Excel avec les données de votre produit, puis notre plateforme créera et remplira les fiches.
- Si vous avez un compte EQUADIS. L’importation est aussi possible.
- Si vous avez un nombre réduit de produits, il reste possible de les créer à la main sur la Plateforme Producteurs (attention les modifications sur la plateforme publique via votre compte professionnel ne sont pas protégées, ces modifications doivent être faites via la plateforme producteurs puis exportées sur la plateforme publique pour obtenir la protection)
→ Pour plus de détails, nous vous invitons à télécharger notre[guide pratique](https://drive.google.com/file/d/1nCWOVzpOmQJs33qXVHVoh0Ugu8LdSspI/view?usp=sharing):-)
