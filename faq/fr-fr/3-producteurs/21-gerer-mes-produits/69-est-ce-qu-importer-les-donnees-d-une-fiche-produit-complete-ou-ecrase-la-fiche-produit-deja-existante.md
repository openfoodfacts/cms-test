Nous utilisons le code barre pour identifier les produits, si vous importez une fiche avec le même code barre qu'une fiche existante, les informations seront fusionnées.Pour les données comme la liste des ingrédients, les valeurs nutritionnelles (une seule valeur correcte possible), les données envoyées via la plateforme producteur écrasent les données existantes. Pour les données comme les labels / catégories / marques (plusieurs valeurs correctes possibles), les informations sont fusionnées. Si des données sont erronées sur la plateforme publique, vous pouvez modifier la fiche produit sur la plateforme publique pour supprimer les valeurs qui ne sont incorrectes.
