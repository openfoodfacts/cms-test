Le but derrière ces différents indicateurs est de donner des informations faciles de compréhension pour le consommateur.

Le Nutri-Score est aujourd’hui rentré dans les mœurs mais son point de vue est purement nutritionnel. C’est pourquoi l’Eco-Score et le groupe NOVA permettent d’apporter des indications complémentaires facilement compréhensibles par le consommateur.
