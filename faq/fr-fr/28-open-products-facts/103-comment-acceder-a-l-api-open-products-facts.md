L'API Open Products Facts est à 98 % identique à l'API Open Food Facts. Si vous consultez la documentation de l'API Open Food Facts, vous devriez avoir une vue d'ensemble claire de l'API Open Products Facts. Nous avons également rédigé un tutoriel dédié pour implémenter l'analyse universelle dans votre application.
