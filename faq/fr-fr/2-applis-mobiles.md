## Comment connaître la version de l'appli que j'utilise ?

Allez dansProfil, puis dansFAQ, puisA propos de cette appli.
Vous devriez voir un numéro de version du type 3.4.5+662. N'oubliez pas de nous indiquer la marque et le modèle du téléphone s'il s'agit de débogage.

---

## L'application ne scanne pas

Cela peut être dû- à un problème réseau
- à un problème de permissions 

- à un problème de code barre

  

Envoyez nous un email via Profil >> Nous Contacter >> Envoyez nous un email.

Toutes les informations utiles devraient être jointes à votre requête.
