---
title: "Quel type d’information dois-je donner pour que le Nutri-Score soit calculé le plus précisément possible ?"
order: 23
lang: fr-fr
category: 8-nutri-score
breadcrumbs: ['/fr-fr/', '/fr-fr/8-nutri-score/']
---

Le calcul du Nutri-Score se fera avec le tableau nutritionnel de votre produit.

N’oubliez pas d’intégrer le pourcentage de fruits, légumes et noix dans votre tableau, afin que le Nutri-Score calculé ne soit pas erroné.
