## Я хотел бы добавить новый логотип для этикеток.

Вот как это происходит:

- Найдите **каноническое название** метки в [таксономии меток](https://github.com/openfoodfacts/openfoodfacts-server/blob/main/taxonomies/labels.txt) (это первый элемент в списке синонимов меток, например, en:100% vegetable)
-

**Используйте логотип хорошего качества**: избегайте использования фотографий авторов, которые не подходят для данного случая; у большинства брендов есть официальные сайты с высококачественными логотипами, иногда в векторном формате (что для нас даже лучше); пока мы используем логотип для объективного информирования о наличии этикетки
на упаковке продукта, запрашивать разрешение не нужно.
----------------------------------------------------------------------

**Назовите файл следующим образом:**: name-of-the-label.[width]x90.png, где width — это ширина логотипа, когда его высота составляет 90 пикселей. Имена файлов должны быть без диакритических знаков, написаны строчными буквами и содержать дефис (-) вместо пробелов.

- Затем добавьте логотип в каталог, соответствующий его каноническому названию. Если каноническое имя — en:something, то они должны быть в /en/. Корневой каталог для файлов logos находится по адресу [https://github.com/openfoodfacts/openfoodfacts-server/tree/main/html/images/lang](https://github.com/openfoodfacts/openfoodfacts-server/tree/main/html/images/lang)
