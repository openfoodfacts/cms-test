## अपलोड केलेल्या फोटोंसाठी कोणता आकार चांगला असेल याबद्दल कागदपत्रांमध्ये कुठेही शिफारसी आहेत का?

नेटवर्क मंद आहे की महाग आहे हे देशांवर अवलंबून असू शकते. ५००० पिक्सेलपेक्षा जास्त वजन किंवा उंची असलेली कोणतीही गोष्ट कदाचित फारशी उपयुक्त नाही. आणि जर तुम्हाला नेटवर्क स्लो असल्याचे कसे तरी कळले, तर २००० पिक्सेलची प्रतिमा देखील उत्तम असेल (प्रतिमा नसण्यापेक्षा निश्चितच चांगली!)

---

## बारकोड नसलेल्या अन्नाचे काय?

ओपन फूड फॅक्ट्समध्ये फक्त पॅकेज्ड फूडबद्दल माहिती असते. उत्पादनांच्या सरासरी मूल्यांसाठी (उदाहरणार्थ, टोमॅटो किंवा केळी) आणि इतर अन्न उत्पादनांसाठी, तुम्ही अधिकृत राष्ट्रीय पोषण डेटाबेसपैकी एक वापरू शकता.

**टीप:** खालील यादीमध्ये काही महत्त्वाचे राष्ट्रीय अन्न डेटाबेस आहेत. जर तुम्हाला वाटत असेल की यादीत इतर काही डेटाबेस समाविष्ट करावा, तर कृपया आमच्याशी येथे संपर्क साधा: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## मी API वापरून अचूक उत्पादन नाव शोधू शकतो का?

दुर्दैवाने, API द्वारे उत्पादनाच्या नावावर सहजपणे आणि अचूकपणे शोधणे अद्याप शक्य झालेले नाही.

श्रेणीनुसार फिल्टर वापरल्याने तुमचा शोध अधिक अचूक होण्यास मदत होऊ शकते.

---

## मी माझ्या प्रकल्पांसाठी डेटा कसा मिळवू/संकलित करू शकतो?

ओपन फूड फॅक्ट्सच्या मुख्य पृष्ठावर, स्क्रीनच्या वरच्या डाव्या कोपऱ्यात, एक स्क्रोलिंग मेनू आहे. त्याच्या तळाशी, तुम्हाला "प्रगत शोध" पर्याय दिसेल, ज्यावर तुम्ही क्लिक करू शकता. त्यानंतर तुमच्या प्रकल्पासाठी कोणते निकष सर्वात संबंधित आहेत हे ठरवणे तुमच्यावर अवलंबून आहे. एकदा निवडल्यानंतर, तुम्ही पृष्ठाच्या तळाशी स्क्रोल करून आणि "निकाल डाउनलोड करा" वर क्लिक करून प्राप्त झालेले निकाल डाउनलोड करू शकाल.

तुम्ही हे देखील पाहू शकता:

- आमचे API दस्तऐवजीकरण:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- ओपन फूड फॅक्ट्सच्या वापराच्या अटी:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- आमच्या डेटावर:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## API वापरण्यासाठी काही अटी आहेत का?

API वापराबद्दलचे सर्व दस्तऐवजीकरण [API दस्तऐवजीकरण पृष्ठ](https://openfoodfacts.github.io/openfoodfacts-server/api/) वर आढळू शकते, परंतु येथे एक संक्षिप्त सारांश आहे:

- ओपन फूड फॅक्ट्स डेटाबेस ओपन डेटाबेस लायसन्स (ODbL) अंतर्गत ओपन डेटा म्हणून उपलब्ध आहे, कायदेशीर तपशीलांसाठी [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) पहा. दोन अटी म्हणजे विशेषता आणि शेअर-अलाईक. जर तुम्ही ओपन फूड फॅक्ट्समधील डेटा इतर डेटाबेससह एकत्रित केला, तर ODbL ला आवश्यक आहे की परिणामी डेटाबेस देखील ओपन डेटा म्हणून रिलीज केला पाहिजे. याचा अर्थ असा की तुम्ही डेटा फक्त अशा स्रोतांसह एकत्र करू शकता जे अशा पुनर्वितरणास अनुमती देतील.

- तुमचा अ‍ॅप ओळखण्यासाठी API कॉल करताना तुम्ही **नेहमी** कस्टम वापरकर्ता-एजंट वापरणे आवश्यक आहे.

- प्रत्येक API एंडपॉइंटसाठी दर-मर्यादा लागू केल्या जातात.

## मी ऐतिहासिक डेटा कसा मिळवू शकतो?

सध्या, आम्ही ऐतिहासिक डेटा डंप (JSONL, MongoDB, CSV) देत नाही.

तथापि, वैयक्तिक उत्पादनांसाठी, API वापरून किंवा उत्पादन पृष्ठावर पुनरावृत्ती वापरून उत्पादन डेटाच्या मागील आवृत्त्यांमध्ये प्रवेश करणे शक्य आहे.

प्रत्येक वेळी उत्पादन अपडेट केले जाते तेव्हा, एक नवीन आवृत्ती (१ पासून सुरू होणारा वाढता अंक) तयार केली जाते.

उदाहरणार्थ, या उत्पादनाची पहिली आवृत्ती (=पहिली उत्पादन आवृत्ती) मिळविण्यासाठी, वापरा

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

त्याचप्रमाणे, rev पॅरामीटर API सह वापरता येतो:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
