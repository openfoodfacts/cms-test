API-ի օգտագործման վերաբերյալ բոլոր փաստաթղթերը կարելի է գտնել API փաստաթղթերի էջում, բայց ահա դրանց համառոտ ամփոփումը.

- Բաց սննդի փաստերի տվյալների բազան հասանելի է որպես բաց տվյալներ Բաց տվյալների բազայի լիցենզիայով (ODbL), իրավական մանրամասների համար տե՛ս https://world.openfoodfacts.org/terms-of-use: Երկու պայմաններն են վերագրումը և համարժեք կիսվելը։ Եթե դուք համատեղում եք Open Food Facts-ի տվյալները այլ տվյալների բազաների հետ, ապա ODbL-ը պահանջում է, որ արդյունքում ստացված տվյալների բազան նույնպես հրապարակվի որպես բաց տվյալներ։ Դա նաև նշանակում է, որ դուք կարող եք տվյալները համատեղել միայն այն աղբյուրների հետ, որոնք թույլ կտան նման վերաբաշխում։

- Դուք միշտ պետք է օգտագործեք հատուկ User-Agent, երբ կատարում եք API կանչեր՝ ձեր ծրագիրը նույնականացնելու համար։

- Յուրաքանչյուր API վերջնակետի համար կիրառվում են արագության սահմանափակումներ։
