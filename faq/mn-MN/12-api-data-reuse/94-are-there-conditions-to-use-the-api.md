API хэрэглээний талаарх бүх баримт бичгийг API баримт бичгийн хуудаснаас олж болох боловч товч тоймыг энд оруулав.

- Нээлттэй Хүнсний Мэдээллийн Сан нь Нээлттэй Мэдээллийн Сангийн Лиценз (ODbL)-ийн дагуу нээлттэй мэдээлэл хэлбэрээр авах боломжтой бөгөөд хууль эрх зүйн дэлгэрэнгүй мэдээллийг https://world.openfoodfacts.org/terms-of-use хаягаар орж үзнэ үү. Хоёр нөхцөл нь хамаарал болон адил тэгш хуваалцах явдал юм. Хэрэв та Open Food Facts-аас авсан өгөгдлийг бусад мэдээллийн сантай нэгтгэвэл ODbL нь үүссэн мэдээллийн санг нээлттэй өгөгдөл болгон гаргахыг шаарддаг. Энэ нь та өгөгдлийг зөвхөн ийм дахин хуваарилалтыг зөвшөөрөх эх сурвалжуудтай нэгтгэж болно гэсэн үг юм.

- Та аппликейшнаа тодорхойлохын тулд API дуудлага хийхдээ үргэлж өөрчлөн тохируулсан Хэрэглэгчийн Агент ашиглах ёстой.

- API төгсгөлийн цэг бүрт ханшийн хязгаарыг мөрддөг.
