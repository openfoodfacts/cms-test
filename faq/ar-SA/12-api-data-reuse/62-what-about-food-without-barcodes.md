لا تحتوي صفحة "حقائق غذائية مفتوحة" إلا على معلومات حول الأطعمة المعلبة. للحصول على القيم المتوسطة للمنتجات الزراعية (على سبيل المثال الطماطم أو الموز) وغيرها من المنتجات الغذائية، يمكنك استخدام إحدى قواعد البيانات الغذائية الوطنية الرسمية بدلاً من ذلك.

ملاحظة: تحتوي القائمة أدناه على بعض أهم قواعد البيانات الغذائية الوطنية. إذا كنت تعتقد أنه ينبغي إضافة قاعدة بيانات أخرى إلى القائمة، فيُرجى التواصل معنا عبر الرابط التالي: https://world.openfoodfacts.org/contact

قائمة قواعد بيانات الأغذية الوطنية

- أستراليا - FSANZ - NUTTAB 2006: https://www.foodstandards.gov.au/media/documents/FSANZ%20Conf%20PostersNUTTAB.pdf
- بلجيكا - NUBEL - بيانات تكوين الأغذية البلجيكية: https://www.internubel.be
- كندا - FCEN: https://aliments-nutrition.canada.ca/cnf-fce/index-fra.jsp
- جمهورية التشيك - قاعدة بيانات تكوين الأغذية في المعهد الوطني للصحة العامة: http://www.chpr.szu.cz/dbdata/foodcomp/nut2001.asp
- الدنمارك - قاعدة بيانات تكوين الأغذية الدنماركية: https://frida.fooddata.dk/?lang=en
- إستونيا - قاعدة بيانات تكوين الأغذية الإستونية: https://tka.nutridata.ee/en/
- فنلندا - قاعدة بيانات تكوين الأغذية الفنلندية - FINELI: https://fineli.fi/fineli/en/index
- فرنسا - CIQUAL: https://www.anses.fr/en/search/site/Table%20ciqual
- ألمانيا - قاعدة بيانات Souci-Fachmann-Kraut على الإنترنت: https://www.sfk.online/#/home أو قاعدة البيانات الألمانية الرسمية: Bundeslebensmittelschlüssel: https://blsdb.de/
- إيطاليا - Banca Dati di Composizione degli Alimenti CREA: https://www.crea.gov.it/web/alimenti-e-nutrizione/banche-dati
- هولندا - قاعدة بيانات تكوين الأغذية الهولندية: https://www.rivm.nl/en/dutch-food-composition-database
- النرويج - جدول التركيب الغذائي النرويجي 2006: https://www.matvaretabellen.no/?language=en
- بولندا - جداول مكونات الأغذية: http://www.izz.waw.pl/en/?lang=en
- إسبانيا - قاعدة بيانات تكوين الأغذية الإسبانية - BEDCA: https://www.bedca.net/bdpub/index.php
- سويسرا - قاعدة بيانات تكوين الأغذية السويسرية: https://www.naehrwertdaten.ch/de/
- المملكة المتحدة - مجموعة بيانات تكوين الأغذية المتكاملة (CoFID): https://www.gov.uk/government/publications/composition-of-foods-integrated-dataset-cofid
- الولايات المتحدة الأمريكية - وزارة الزراعة الأمريكية: https://ndb.nal.usda.gov/
