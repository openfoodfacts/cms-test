يمكنك العثور على جميع الوثائق المتعلقة باستخدام واجهة برمجة التطبيقات (API) في صفحة وثائق واجهة برمجة التطبيقات، ولكن إليك ملخصًا سريعًا:

- قاعدة بيانات Open Food Facts متاحة كبيانات مفتوحة بموجب ترخيص قاعدة البيانات المفتوحة (ODbL)، راجع https://world.openfoodfacts.org/terms-of-use للاطلاع على التفاصيل القانونية. الشرطان هما الإسناد والمشاركة المتساوية. إذا قمت بدمج البيانات من Open Food Facts مع قواعد بيانات أخرى، فإن ODbL يتطلب أن يتم إصدار قاعدة البيانات الناتجة كبيانات مفتوحة أيضًا. وهذا يعني أيضاً أنه لا يمكنك دمج البيانات إلا مع المصادر التي تسمح بإعادة التوزيع هذه.

- يجب عليك دائمًا استخدام وكيل مستخدم مخصص عند إجراء استدعاءات واجهة برمجة التطبيقات لتحديد تطبيقك.

- يتم تطبيق حدود المعدل لكل نقطة نهاية API.
