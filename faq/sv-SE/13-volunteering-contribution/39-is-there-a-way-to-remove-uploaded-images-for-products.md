Endast moderatorer kan ta bort foton för att undvika potentiell vandalism.

Fråga bara på Slack eller på contact@openfoodfacts.org om du vill ta bort dubbletter eller olämpliga foton (försök att ange streckkodsnumret eller URL:en).

Vi har också ett nytt API för bildrapporter om du är programmerare.
