## Оё дар ҳуҷҷатҳо тавсияҳо дар бораи андозаи мувофиқ барои аксҳои боршуда мавҷуданд?

Ин метавонад аз кишварҳо вобаста бошад, агар шабака суст ё гарон бошад. ҳар чизе, ки аз 5000 пиксел вазн ё қад болотар аст, эҳтимолан чандон муфид нест. ва агар шумо ба ягон роҳ муайян карда тавонед, ки шабака суст кор мекунад, пас ҳатто як тасвири 2000 пикселӣ хеле хуб мебуд (албатта аз надоштани тасвир беҳтар аст!)

## Дар бораи хӯроке, ки штрих-код надорад, чӣ гуфтан мумкин аст?

Маълумоти кушодаи хӯрокворӣ танҳо маълумотро дар бораи хӯрокҳои бастабандишуда дар бар мегирад. Барои арзишҳои миёнаи маҳсулот (масалан, помидор ё банан) ва дигар маҳсулоти хӯрокворӣ, шумо метавонед яке аз пойгоҳҳои додаҳои расмии миллии ғизоро истифода баред.

**Эзоҳ:** Рӯйхати зер баъзе аз муҳимтарин пойгоҳҳои миллии озуқавориро дар бар мегирад. Агар шумо фикр кунед, ки ягон пойгоҳи додаҳои дигар бояд ба рӯйхат дохил карда шаванд, лутфан бо мо тамос гиред: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## Оё ман метавонам номи дақиқи маҳсулотро бо API ҷустуҷӯ кунам?

Мутаассифона, ҳоло танҳо бо номи маҳсулот ва дақиқ тавассути API ҷустуҷӯи осон имконнопазир аст.

Истифодаи филтр дар категория метавонад ба шумо дар дақиқтар кардани ҷустуҷӯ кӯмак кунад.

---

## Чӣ тавр ман метавонам ба маълумот барои лоиҳаҳои худ дастрасӣ/ҷамъоварӣ кунам?

Дар саҳифаи асосии Open Food Facts, дар кунҷи чапи болоии экран, менюи ҳаракаткунанда мавҷуд аст. Дар поёни он шумо варианти "ҷустуҷӯи пешрафта"-ро хоҳед ёфт, ки шумо метавонед онро клик кунед. Пас, муайян кардани он ки кадом меъёрҳо барои лоиҳа(ҳо)-и шумо аз ҳама мувофиқтаранд, аз шумо вобаста аст. Пас аз интихоб, шумо метавонед натиҷаҳои бадастовардаро тавассути ҳаракат дар поёни саҳифа ва пахш кардани тугмаи "Натиҷаҳоро зеркашӣ кунед" зеркашӣ кунед.

Шумо инчунин метавонед бо инҳо машварат кунед:

- Ҳуҷҷатҳои API-и мо:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Шартҳои истифодаи Open Food Facts:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Дар маълумоти мо:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## Оё шароит барои истифодаи API вуҷуд дорад?

Ҳама ҳуҷҷатҳо дар бораи истифодаи API-ро дар [саҳифаи ҳуҷҷатҳои API](https://openfoodfacts.github.io/openfoodfacts-server/api/) пайдо кардан мумкин аст, аммо дар ин ҷо хулосаи кӯтоҳ оварда шудааст:

- Пойгоҳи додаҳои Open Food Facts ҳамчун маълумоти кушода тибқи Литсензияи Open Position Database (ODbL) дастрас аст, барои тафсилоти ҳуқуқӣ ба [https://world.openfoodfacts.org/terms-of-use[https://world.openfoodfacts.org/terms-of-use] нигаред. Ин ду шарт мансубият ва тақсими якхела мебошанд. Агар шумо маълумотро аз Open Food Facts бо дигар пойгоҳҳои додаҳо якҷоя кунед, пас ODbL талаб мекунад, ки пойгоҳи додаҳои натиҷавӣ низ ҳамчун маълумоти кушода нашр карда шаванд. Ин инчунин маънои онро дорад, ки шумо метавонед маълумотро танҳо бо манбаъҳое муттаҳид кунед, ки чунин тақсимоти такрориро имконпазир мегардонанд.

- Ҳангоми иҷрои зангҳои API барои муайян кардани барномаи худ, шумо бояд **ҳамеша** аз агенти корбарии корбар истифода баред.

- Маҳдудиятҳои суръат барои ҳар як нуқтаи ниҳоии API татбиқ карда мешаванд.

## Чӣ тавр ман метавонам ба маълумоти таърихӣ дастрасӣ пайдо кунам?

Айни замон, мо нусхабардории маълумоти таърихиро (JSONL, MongoDB, CSV) пешниҳод намекунем.

Аммо, барои маҳсулоти алоҳида, дастрасӣ ба версияҳои қаблии маълумоти маҳсулот бо истифода аз API ё дар саҳифаи маҳсулот бо истифода аз тағйирот имконпазир аст.

Ҳар дафъае, ки маҳсулот навсозӣ мешавад, як таҷдиди назар (рақами афзоянда аз 1 сар мешавад) эҷод карда мешавад.

Масалан, барои гирифтани нусхаи аввали ин маҳсулот (=версияи аввалини маҳсулот), истифода баред

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Ба ҳамин монанд, параметри rev метавонад бо API истифода шавад:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
