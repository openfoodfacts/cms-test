Open Pet Food Facts API 与 Open Food Facts API 有 98% 的相似度，因此，如果您使用 Open Food Facts API 文档，您应该对 Open Pet Food Facts API 有一个相当清晰的了解。

请注意，知识面板、生态评分、营养评分等功能不受支持（可能是由于主题原因，也可能是由于代码库较旧）。

https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/
