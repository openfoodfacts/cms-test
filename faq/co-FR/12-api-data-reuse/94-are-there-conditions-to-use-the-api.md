Tutta a ducumentazione nantu à l'usu di l'API si pò truvà nantu à a pagina di ducumentazione di l'API, ma eccu un riassuntu rapidu:

- A basa di dati Open Food Facts hè dispunibule cum'è dati aperti sottu a Licenza Open Database (ODbL), vede https://world.openfoodfacts.org/terms-of-use per i dettagli legali. E duie cundizioni sò l'attribuzione è a spartera in listessu modu. Sè vo cumminate dati da Open Food Facts cù altre basi di dati, allora l'ODbL richiede chì a basa di dati risultante sia ancu publicata cum'è dati aperti. Significa ancu chì pudete cumminà i dati solu cù fonti chì permettenu una tale ridistribuzione.

- Duvete sempre aduprà un User-Agent persunalizatu quandu eseguite chjamate API per identificà a vostra app.

- I limiti di velocità sò applicati per ogni endpoint API.
