Open Food Facts só contén información sobre alimentos envasados. Para os valores medios de produtos agrícolas (por exemplo, tomates ou plátanos) e outros produtos alimenticios, podes usar unha das bases de datos nacionais oficiais de nutrición.

Nota: A lista que aparece a continuación contén algunhas das bases de datos nacionais de alimentos máis importantes. Se cres que se debería incluír algunha outra base de datos na lista, ponte en contacto connosco en: https://world.openfoodfacts.org/contact

Lista de bases de datos nacionais de alimentos

- Australia - FSANZ - NUTTAB 2006: https://www.foodstandards.gov.au/media/documents/FSANZ%20Conf%20PostersNUTTAB.pdf
- Bélxica - NUBEL - Datos belgas de composición de alimentos: https://www.internubel.be
- Canadá - FCEN: https://aliments-nutrition.canada.ca/cnf-fce/index-fra.jsp
- República Checa - Base de datos de composición de alimentos no Instituto Nacional de Saúde Pública: http://www.chpr.szu.cz/dbdata/foodcomp/nut2001.asp
- Dinamarca - Banco de datos danés de composición de alimentos: https://frida.fooddata.dk/?lang=en
- Estonia - Base de datos estoniana de composición de alimentos: https://tka.nutridata.ee/en/
- Finlandia - Base de datos finlandesa de composición de alimentos - FINELI: https://fineli.fi/fineli/en/index
- Francia - CIQUAL: https://www.anses.fr/en/search/site/Table%20ciqual
- Alemaña - Base de datos en liña Souci-Fachmann-Kraut: https://www.sfk.online/#/home ou a base de datos oficial alemá: Bundeslebensmittelschlüssel: https://blsdb.de/
- Italia - Banca Dati di Composizione degli Alimenti CREA: https://www.crea.gov.it/web/alimenti-e-nutrizione/banche-dati
- Países Baixos - Base de datos neerlandesa de composición de alimentos: https://www.rivm.nl/en/dutch-food-composition-database
- Noruega - Táboa de composición de alimentos norueguesa de 2006: https://www.matvaretabellen.no/?language=en
- Polonia - Táboas de composición de alimentos: http://www.izz.waw.pl/en/?lang=en
- España - Base de datos española de composición de alimentos - BEDCA: https://www.bedca.net/bdpub/index.php
- Suíza - Base de datos suíza de composición de alimentos: https://www.naehrwertdaten.ch/de/
- Reino Unido - Conxunto de datos integrado sobre a composición dos alimentos (CoFID): https://www.gov.uk/government/publications/composition-of-foods-integrated-dataset-cofid
- EUA - USDA: https://ndb.nal.usda.gov/
