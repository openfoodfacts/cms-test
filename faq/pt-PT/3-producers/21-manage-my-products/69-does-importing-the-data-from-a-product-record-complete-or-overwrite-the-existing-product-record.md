Utilizamos o código de barras para identificar os produtos. Se importar um registo com o mesmo código de barras de um registo existente, as informações serão combinadas.

Para dados como a lista de ingredientes e os valores nutricionais (apenas é possível um valor correto), os dados enviados pela plataforma do produtor substituirão os dados existentes.

Para dados como etiquetas/categorias/marcas (vários valores corretos possíveis), as informações são fundidas. Caso os dados estejam incorretos na plataforma pública, pode modificar a ficha do produto na plataforma pública para remover os valores incorretos.
