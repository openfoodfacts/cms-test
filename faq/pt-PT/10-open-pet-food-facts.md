## Porque é que não existe um Nutri-Score para animais?

Cada animal é diferente (uma cobra não tem as mesmas necessidades nutricionais que um gato ou um cão).

Mesmo dentro da mesma espécie animal, como os cães, as necessidades nutricionais variam de acordo com o estilo de vida sedentário, uma possível castração ou doenças.

Por isso, é complicado calcular uma pontuação para a alimentação animal sem envolver dezenas de investigadores e criar dezenas de algoritmos diferentes, o que ainda não foi feito.

## Where can I find the Open Pet Food Facts API ?

Open Pet Food Facts API is 98% the same as Open Food Facts API so if you use Open Food Facts API documentation you should have a pretty clear view of the Open Pet Food Facts API.

Note that features like the Knowledge Panels, Eco-Score, Nutri-Score are not supported (either due to the topic, or to the old codebase).

[https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/](https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/)
