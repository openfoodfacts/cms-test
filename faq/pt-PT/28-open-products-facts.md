## Como aceder ao Open Products Facts no iPhone?

O Open Products Facts está disponível na aplicação principal Open Food Facts. Esta integração permite que os utilizadores de iPhone acedam a ambos os bancos de dados através de um único aplicativo prático.

---

## Gostaria de apoiar a Open Products Facts. O que posso fazer ?

Por favor, leia a nossa página wiki, listamos formas de se envolver: [https://wiki.openfoodfacts.org/Open_Products_Facts](https://wiki.openfoodfacts.org/Open_Products_Facts)

Para apoiar o Open Products Facts, pode considerar as seguintes opções, cada uma adaptada a diferentes tipos de contribuições:

-

**Patrocinadores filantrópicos/estatais para o Open Products Facts**:
As contribuições filantrópicas são cruciais para concretizar a visão de uma plataforma abrangente de dados circulares, pontuação de produtos e agregação de soluções circulares, em mais de 20.000 categorias, um objetivo fundamental do Open Products Facts. Muitos componentes essenciais desta visão ainda não foram financiados, e o seu apoio financeiro poderá ter um impacto significativo. Os seus donativos podem ajudar a financiar projetos e infraestruturas essenciais. Para contribuir financeiramente, por favor contacte-nos através do endereço contact@openfoodfacts.org.

-

**Colaboradores de dados que utilizam a aplicação Open Products Facts**:
As contribuições de dados são vitais para a expansão dos Open Products Facts. Ao utilizar a aplicação Android para adicionar novas informações de produtos, atualizar entradas existentes ou verificar dados, ajuda a expandir e a melhorar a base de dados. Este esforço coletivo contribui para a construção de um recurso robusto e fiável para todos.

-

**Colaboradores técnicos do Open Products Facts**:
Se possui competências técnicas em áreas como programação, análise de dados ou design de experiência do utilizador, a sua experiência pode beneficiar significativamente o Open Products Facts. Participar no desenvolvimento da aplicação, melhorar a sua funcionalidade ou melhorar as interfaces do utilizador são contribuições valiosas. Para participar, junte-se ao canal #openproductsfacts no Slack, onde colabora a comunidade Open Products Facts. Pode encontrar um link para este espaço de trabalho Slack no site Open Food Facts.

Cada tipo de contribuição é importante para o crescimento e sucesso do Open Products Facts, ajudando a criar uma plataforma mais sustentável e informativa para todos.

## Onde posso encontrar a API Open Products Facts?

A API Open Products Facts é 98% igual à API Open Food Facts, pelo que, se utilizar a documentação da API Open Food Facts, terá uma visão bastante clara da API Open Products Facts. Também elaborámos um tutorial específico para permitir que implemente a leitura universal na sua aplicação.

[https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/](https://openfoodfacts.github.io/openfoodfacts-server/api/ref-v2/)
