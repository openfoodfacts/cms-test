Toda a documentação sobre a utilização da API pode ser encontrada na página de documentação da API, mas aqui está um breve resumo:

- A base de dados Open Food Facts está disponível como dados abertos sob a Licença de Base de Dados Aberta (ODbL). Consulte https://world.openfoodfacts.org/terms-of-use para obter detalhes legais. As duas condições são a atribuição e a partilha igualitária. Se combinar os dados do Open Food Facts com outras bases de dados, o ODbL exige que a base de dados resultante também seja disponibilizada como dados abertos. Significa também que só pode combinar os dados com fontes que permitam essa redistribuição.

- Deve sempre utilizar um User-Agent personalizado ao realizar chamadas de API para identificar a sua aplicação.

- Os limites de taxa são aplicados a cada endpoint da API.
