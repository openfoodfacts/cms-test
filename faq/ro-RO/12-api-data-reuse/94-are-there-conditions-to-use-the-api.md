Toată documentația despre utilizarea API-ului poate fi găsită pe pagina de documentație API, dar iată un scurt rezumat:

- Baza de date Open Food Facts este disponibilă ca date deschise sub licența Open Database License (ODbL), consultați https://world.openfoodfacts.org/terms-of-use pentru detalii legale. Cele două condiții sunt atribuirea și distribuirea în mod similar. Dacă combinați date din Open Food Facts cu alte baze de date, atunci ODbL impune ca baza de date rezultată să fie publicată și ca date deschise. De asemenea, înseamnă că puteți combina datele doar cu surse care ar permite o astfel de redistribuire.

- Trebuie să utilizați întotdeauna un User-Agent personalizat atunci când efectuați apeluri API pentru a identifica aplicația.

- Limitele de rată sunt impuse pentru fiecare punct final API.
