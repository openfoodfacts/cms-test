## ਕੀ ਦਸਤਾਵੇਜ਼ਾਂ ਵਿੱਚ ਕਿਤੇ ਵੀ ਇਸ ਬਾਰੇ ਸਿਫ਼ਾਰਸ਼ਾਂ ਹਨ ਕਿ ਅਪਲੋਡ ਕੀਤੀਆਂ ਫੋਟੋਆਂ ਲਈ ਕਿਹੜਾ ਆਕਾਰ ਚੰਗਾ ਹੋਵੇਗਾ?

ਇਹ ਦੇਸ਼ਾਂ 'ਤੇ ਨਿਰਭਰ ਕਰ ਸਕਦਾ ਹੈ, ਜੇਕਰ ਨੈੱਟਵਰਕ ਹੌਲੀ ਹੈ ਜਾਂ ਮਹਿੰਗਾ ਹੈ। 5000 ਪਿਕਸਲ ਤੋਂ ਵੱਧ ਭਾਰ ਜਾਂ ਉਚਾਈ ਸ਼ਾਇਦ ਬਹੁਤ ਉਪਯੋਗੀ ਨਹੀਂ ਹੈ। ਅਤੇ ਜੇਕਰ ਤੁਸੀਂ ਕਿਸੇ ਤਰ੍ਹਾਂ ਪਤਾ ਲਗਾ ਸਕਦੇ ਹੋ ਕਿ ਨੈੱਟਵਰਕ ਹੌਲੀ ਹੈ, ਤਾਂ 2000 ਪਿਕਸਲ ਦੀ ਤਸਵੀਰ ਵੀ ਬਹੁਤ ਵਧੀਆ ਹੋਵੇਗੀ (ਨਿਸ਼ਚਤ ਤੌਰ 'ਤੇ ਤਸਵੀਰ ਨਾ ਹੋਣ ਨਾਲੋਂ ਬਿਹਤਰ!)

## ਬਾਰਕੋਡ ਤੋਂ ਬਿਨਾਂ ਭੋਜਨ ਬਾਰੇ ਕੀ?

ਓਪਨ ਫੂਡ ਫੈਕਟਸ ਵਿੱਚ ਸਿਰਫ਼ ਪੈਕ ਕੀਤੇ ਭੋਜਨ ਬਾਰੇ ਜਾਣਕਾਰੀ ਹੁੰਦੀ ਹੈ। ਔਸਤਨ ਪੈਦਾਵਾਰ (ਉਦਾਹਰਨ ਲਈ, ਟਮਾਟਰ ਜਾਂ ਕੇਲੇ) ਅਤੇ ਹੋਰ ਭੋਜਨ ਉਤਪਾਦਾਂ ਲਈ, ਤੁਸੀਂ ਇਸਦੀ ਬਜਾਏ ਅਧਿਕਾਰਤ ਰਾਸ਼ਟਰੀ ਪੋਸ਼ਣ ਡੇਟਾਬੇਸ ਵਿੱਚੋਂ ਇੱਕ ਦੀ ਵਰਤੋਂ ਕਰ ਸਕਦੇ ਹੋ।

**ਨੋਟ:** ਹੇਠਾਂ ਦਿੱਤੀ ਸੂਚੀ ਵਿੱਚ ਕੁਝ ਸਭ ਤੋਂ ਮਹੱਤਵਪੂਰਨ ਰਾਸ਼ਟਰੀ ਭੋਜਨ ਡੇਟਾਬੇਸ ਸ਼ਾਮਲ ਹਨ। ਜੇਕਰ ਤੁਹਾਨੂੰ ਲੱਗਦਾ ਹੈ ਕਿ ਸੂਚੀ ਵਿੱਚ ਕੋਈ ਹੋਰ ਡੇਟਾਬੇਸ ਸ਼ਾਮਲ ਕੀਤਾ ਜਾਣਾ ਚਾਹੀਦਾ ਹੈ, ਤਾਂ ਕਿਰਪਾ ਕਰਕੇ ਸਾਡੇ ਨਾਲ ਇੱਥੇ ਸੰਪਰਕ ਕਰੋ: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

## ਕੀ ਮੈਂ API ਨਾਲ ਇੱਕ ਸਟੀਕ ਉਤਪਾਦ ਨਾਮ ਖੋਜ ਸਕਦਾ ਹਾਂ?

ਬਦਕਿਸਮਤੀ ਨਾਲ, API ਰਾਹੀਂ ਸਿਰਫ਼ ਉਤਪਾਦ ਦੇ ਨਾਮ 'ਤੇ ਆਸਾਨੀ ਨਾਲ ਅਤੇ ਸਹੀ ਢੰਗ ਨਾਲ ਖੋਜ ਕਰਨਾ ਅਜੇ ਸੰਭਵ ਨਹੀਂ ਹੈ।

ਹਾਲਾਂਕਿ, ਸ਼੍ਰੇਣੀ 'ਤੇ ਫਿਲਟਰ ਦੀ ਵਰਤੋਂ ਕਰਨ ਨਾਲ ਤੁਹਾਡੀ ਖੋਜ ਨੂੰ ਹੋਰ ਸਟੀਕ ਬਣਾਉਣ ਵਿੱਚ ਮਦਦ ਮਿਲ ਸਕਦੀ ਹੈ।

## ਮੈਂ ਆਪਣੇ ਪ੍ਰੋਜੈਕਟਾਂ ਲਈ ਡੇਟਾ ਕਿਵੇਂ ਪ੍ਰਾਪਤ/ਇਕੱਠਾ ਕਰ ਸਕਦਾ/ਸਕਦੀ ਹਾਂ?

ਓਪਨ ਫੂਡ ਫੈਕਟਸ ਦੇ ਮੁੱਖ ਪੰਨੇ 'ਤੇ, ਸਕ੍ਰੀਨ ਦੇ ਉੱਪਰ ਖੱਬੇ ਕੋਨੇ 'ਤੇ, ਇੱਕ ਸਕ੍ਰੋਲਿੰਗ ਮੀਨੂ ਹੈ। ਇਸਦੇ ਹੇਠਾਂ, ਤੁਹਾਨੂੰ "ਐਡਵਾਂਸਡ ਸਰਚ" ਵਿਕਲਪ ਮਿਲੇਗਾ, ਜਿਸ 'ਤੇ ਤੁਸੀਂ ਕਲਿੱਕ ਕਰ ਸਕਦੇ ਹੋ। ਫਿਰ ਇਹ ਤੁਹਾਡੇ 'ਤੇ ਨਿਰਭਰ ਕਰਦਾ ਹੈ ਕਿ ਤੁਹਾਡੇ ਪ੍ਰੋਜੈਕਟ(ਪ੍ਰੋਜੈਕਟਾਂ) ਲਈ ਕਿਹੜੇ ਮਾਪਦੰਡ ਸਭ ਤੋਂ ਵੱਧ ਢੁਕਵੇਂ ਹਨ। ਇੱਕ ਵਾਰ ਚੁਣਨ ਤੋਂ ਬਾਅਦ, ਤੁਸੀਂ ਪੰਨੇ ਦੇ ਹੇਠਾਂ ਸਕ੍ਰੌਲ ਕਰਕੇ ਅਤੇ "ਨਤੀਜੇ ਡਾਊਨਲੋਡ ਕਰੋ" 'ਤੇ ਕਲਿੱਕ ਕਰਕੇ ਪ੍ਰਾਪਤ ਨਤੀਜਿਆਂ ਨੂੰ ਡਾਊਨਲੋਡ ਕਰਨ ਦੇ ਯੋਗ ਹੋਵੋਗੇ।

ਤੁਸੀਂ ਇਹਨਾਂ ਨਾਲ ਵੀ ਸਲਾਹ ਲੈ ਸਕਦੇ ਹੋ:

- ਸਾਡਾ API ਦਸਤਾਵੇਜ਼:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- ਓਪਨ ਫੂਡ ਫੈਕਟਸ ਦੀ ਵਰਤੋਂ ਦੀਆਂ ਸ਼ਰਤਾਂ:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- ਸਾਡੇ ਡੇਟਾ 'ਤੇ:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## ਕੀ API ਦੀ ਵਰਤੋਂ ਕਰਨ ਲਈ ਕੋਈ ਸ਼ਰਤਾਂ ਹਨ?

API ਵਰਤੋਂ ਬਾਰੇ ਸਾਰੇ ਦਸਤਾਵੇਜ਼ [API ਦਸਤਾਵੇਜ਼ ਪੰਨੇ](https://openfoodfacts.github.io/openfoodfacts-server/api/) 'ਤੇ ਮਿਲ ਸਕਦੇ ਹਨ, ਪਰ ਇੱਥੇ ਇੱਕ ਛੋਟਾ ਜਿਹਾ ਸਾਰ ਹੈ:

- ਓਪਨ ਫੂਡ ਫੈਕਟਸ ਡੇਟਾਬੇਸ ਓਪਨ ਡੇਟਾਬੇਸ ਲਾਇਸੈਂਸ (ODbL) ਦੇ ਤਹਿਤ ਓਪਨ ਡੇਟਾ ਦੇ ਰੂਪ ਵਿੱਚ ਉਪਲਬਧ ਹੈ, ਕਾਨੂੰਨੀ ਵੇਰਵਿਆਂ ਲਈ [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) ਵੇਖੋ। ਦੋ ਸ਼ਰਤਾਂ ਹਨ ਵਿਸ਼ੇਸ਼ਤਾ ਅਤੇ ਸਾਂਝਾ-ਇੱਕੋ ਜਿਹਾ। ਜੇਕਰ ਤੁਸੀਂ ਓਪਨ ਫੂਡ ਫੈਕਟਸ ਦੇ ਡੇਟਾ ਨੂੰ ਹੋਰ ਡੇਟਾਬੇਸਾਂ ਨਾਲ ਜੋੜਦੇ ਹੋ, ਤਾਂ ODbL ਇਹ ਮੰਗ ਕਰਦਾ ਹੈ ਕਿ ਨਤੀਜੇ ਵਜੋਂ ਡੇਟਾਬੇਸ ਨੂੰ ਓਪਨ ਡੇਟਾ ਦੇ ਰੂਪ ਵਿੱਚ ਵੀ ਜਾਰੀ ਕੀਤਾ ਜਾਣਾ ਚਾਹੀਦਾ ਹੈ। ਇਸਦਾ ਇਹ ਵੀ ਮਤਲਬ ਹੈ ਕਿ ਤੁਸੀਂ ਡੇਟਾ ਨੂੰ ਸਿਰਫ਼ ਉਹਨਾਂ ਸਰੋਤਾਂ ਨਾਲ ਜੋੜ ਸਕਦੇ ਹੋ ਜੋ ਅਜਿਹੇ ਪੁਨਰਵੰਡਨ ਦੀ ਆਗਿਆ ਦੇਣਗੇ।

- ਤੁਹਾਨੂੰ ਆਪਣੀ ਐਪ ਦੀ ਪਛਾਣ ਕਰਨ ਲਈ API ਕਾਲਾਂ ਕਰਦੇ ਸਮੇਂ **ਹਮੇਸ਼ਾ** ਇੱਕ ਕਸਟਮ ਯੂਜ਼ਰ-ਏਜੰਟ ਦੀ ਵਰਤੋਂ ਕਰਨੀ ਚਾਹੀਦੀ ਹੈ।

- ਹਰੇਕ API ਐਂਡਪੁਆਇੰਟ ਲਈ ਦਰ-ਸੀਮਾਵਾਂ ਲਾਗੂ ਕੀਤੀਆਂ ਜਾਂਦੀਆਂ ਹਨ।

## ਮੈਂ ਇਤਿਹਾਸਕ ਡੇਟਾ ਤੱਕ ਕਿਵੇਂ ਪਹੁੰਚ ਕਰ ਸਕਦਾ ਹਾਂ?

ਵਰਤਮਾਨ ਵਿੱਚ, ਅਸੀਂ ਇਤਿਹਾਸਕ ਡੇਟਾ ਡੰਪ (JSONL, MongoDB, CSV) ਦੀ ਪੇਸ਼ਕਸ਼ ਨਹੀਂ ਕਰਦੇ ਹਾਂ।

ਹਾਲਾਂਕਿ, ਵਿਅਕਤੀਗਤ ਉਤਪਾਦਾਂ ਲਈ, API ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਜਾਂ ਉਤਪਾਦ ਪੰਨੇ 'ਤੇ ਸੰਸ਼ੋਧਨਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਉਤਪਾਦ ਡੇਟਾ ਦੇ ਪਿਛਲੇ ਸੰਸਕਰਣਾਂ ਤੱਕ ਪਹੁੰਚ ਕਰਨਾ ਸੰਭਵ ਹੈ।

ਹਰ ਵਾਰ ਜਦੋਂ ਕੋਈ ਉਤਪਾਦ ਅੱਪਡੇਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ, ਤਾਂ ਇੱਕ ਨਵਾਂ ਸੰਸ਼ੋਧਨ (1 ਤੋਂ ਸ਼ੁਰੂ ਹੋਣ ਵਾਲਾ ਵਧਦਾ ਅੰਕ) ਬਣਾਇਆ ਜਾਂਦਾ ਹੈ।

ਉਦਾਹਰਨ ਲਈ, ਇਸ ਉਤਪਾਦ ਦਾ ਪਹਿਲਾ ਸੰਸ਼ੋਧਨ (=ਪਹਿਲਾ ਉਤਪਾਦ ਸੰਸਕਰਣ) ਪ੍ਰਾਪਤ ਕਰਨ ਲਈ, ਵਰਤੋਂ

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).
