ਓਪਨ ਪੇਟ ਫੂਡ ਫੈਕਟਸ API 98% ਓਪਨ ਫੂਡ ਫੈਕਟਸ API ਦੇ ਸਮਾਨ ਹੈ, ਇਸ ਲਈ ਜੇਕਰ ਤੁਸੀਂ ਓਪਨ ਫੂਡ ਫੈਕਟਸ API ਦਸਤਾਵੇਜ਼ਾਂ ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹੋ ਤਾਂ ਤੁਹਾਡੇ ਕੋਲ ਓਪਨ ਪੇਟ ਫੂਡ ਫੈਕਟਸ API ਦਾ ਇੱਕ ਸਪਸ਼ਟ ਦ੍ਰਿਸ਼ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ।

ਧਿਆਨ ਦਿਓ ਕਿ ਗਿਆਨ ਪੈਨਲ, ਈਕੋ-ਸਕੋਰ, ਨਿਊਟਰੀ-ਸਕੋਰ ਵਰਗੀਆਂ ਵਿਸ਼ੇਸ਼ਤਾਵਾਂ ਸਮਰਥਿਤ ਨਹੀਂ ਹਨ (ਜਾਂ ਤਾਂ ਵਿਸ਼ੇ ਦੇ ਕਾਰਨ, ਜਾਂ ਪੁਰਾਣੇ ਕੋਡਬੇਸ ਦੇ ਕਾਰਨ)।
