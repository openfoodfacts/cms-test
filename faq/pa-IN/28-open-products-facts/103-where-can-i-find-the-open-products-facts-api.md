ਓਪਨ ਪ੍ਰੋਡਕਟਸ ਫੈਕਟਸ API 98% ਓਪਨ ਫੂਡ ਫੈਕਟਸ API ਦੇ ਸਮਾਨ ਹੈ, ਇਸ ਲਈ ਜੇਕਰ ਤੁਸੀਂ ਓਪਨ ਫੂਡ ਫੈਕਟਸ API ਦਸਤਾਵੇਜ਼ਾਂ ਦੀ ਵਰਤੋਂ ਕਰਦੇ ਹੋ ਤਾਂ ਤੁਹਾਡੇ ਕੋਲ ਓਪਨ ਪ੍ਰੋਡਕਟਸ ਫੈਕਟਸ API ਦਾ ਇੱਕ ਸਪਸ਼ਟ ਦ੍ਰਿਸ਼ ਹੋਣਾ ਚਾਹੀਦਾ ਹੈ। ਅਸੀਂ ਤੁਹਾਡੀ ਐਪ ਵਿੱਚ ਯੂਨੀਵਰਸਲ ਸਕੈਨ ਲਾਗੂ ਕਰਨ ਲਈ ਸਮਰਪਿਤ ਟਿਊਟੋਰਿਅਲ ਵੀ ਲਿਖਿਆ ਹੈ।
