ਬਦਕਿਸਮਤੀ ਨਾਲ, API ਰਾਹੀਂ ਸਿਰਫ਼ ਉਤਪਾਦ ਦੇ ਨਾਮ 'ਤੇ ਆਸਾਨੀ ਨਾਲ ਅਤੇ ਸਹੀ ਢੰਗ ਨਾਲ ਖੋਜ ਕਰਨਾ ਅਜੇ ਸੰਭਵ ਨਹੀਂ ਹੈ।

ਹਾਲਾਂਕਿ, ਸ਼੍ਰੇਣੀ 'ਤੇ ਫਿਲਟਰ ਦੀ ਵਰਤੋਂ ਕਰਨ ਨਾਲ ਤੁਹਾਡੀ ਖੋਜ ਨੂੰ ਹੋਰ ਸਟੀਕ ਬਣਾਉਣ ਵਿੱਚ ਮਦਦ ਮਿਲ ਸਕਦੀ ਹੈ।
