API ਵਰਤੋਂ ਬਾਰੇ ਸਾਰੇ ਦਸਤਾਵੇਜ਼ API ਦਸਤਾਵੇਜ਼ ਪੰਨੇ 'ਤੇ ਮਿਲ ਸਕਦੇ ਹਨ, ਪਰ ਇੱਥੇ ਇੱਕ ਛੋਟਾ ਜਿਹਾ ਸਾਰ ਹੈ:

- ਓਪਨ ਫੂਡ ਫੈਕਟਸ ਡੇਟਾਬੇਸ ਓਪਨ ਡੇਟਾਬੇਸ ਲਾਇਸੈਂਸ (ODbL) ਦੇ ਤਹਿਤ ਓਪਨ ਡੇਟਾ ਦੇ ਰੂਪ ਵਿੱਚ ਉਪਲਬਧ ਹੈ, ਕਾਨੂੰਨੀ ਵੇਰਵਿਆਂ ਲਈ https://world.openfoodfacts.org/terms-of-use ਵੇਖੋ। ਦੋ ਸ਼ਰਤਾਂ ਹਨ ਵਿਸ਼ੇਸ਼ਤਾ ਅਤੇ ਸਾਂਝਾ-ਇੱਕੋ ਜਿਹਾ। ਜੇਕਰ ਤੁਸੀਂ ਓਪਨ ਫੂਡ ਫੈਕਟਸ ਦੇ ਡੇਟਾ ਨੂੰ ਹੋਰ ਡੇਟਾਬੇਸਾਂ ਨਾਲ ਜੋੜਦੇ ਹੋ, ਤਾਂ ODbL ਇਹ ਮੰਗ ਕਰਦਾ ਹੈ ਕਿ ਨਤੀਜੇ ਵਜੋਂ ਡੇਟਾਬੇਸ ਨੂੰ ਓਪਨ ਡੇਟਾ ਦੇ ਰੂਪ ਵਿੱਚ ਵੀ ਜਾਰੀ ਕੀਤਾ ਜਾਣਾ ਚਾਹੀਦਾ ਹੈ। ਇਸਦਾ ਇਹ ਵੀ ਮਤਲਬ ਹੈ ਕਿ ਤੁਸੀਂ ਡੇਟਾ ਨੂੰ ਸਿਰਫ਼ ਉਹਨਾਂ ਸਰੋਤਾਂ ਨਾਲ ਜੋੜ ਸਕਦੇ ਹੋ ਜੋ ਅਜਿਹੇ ਪੁਨਰਵੰਡਨ ਦੀ ਆਗਿਆ ਦੇਣਗੇ।

- ਤੁਹਾਨੂੰ ਆਪਣੀ ਐਪ ਦੀ ਪਛਾਣ ਕਰਨ ਲਈ API ਕਾਲਾਂ ਕਰਦੇ ਸਮੇਂ ਹਮੇਸ਼ਾਂ ਇੱਕ ਕਸਟਮ ਯੂਜ਼ਰ-ਏਜੰਟ ਦੀ ਵਰਤੋਂ ਕਰਨੀ ਚਾਹੀਦੀ ਹੈ।

- ਹਰੇਕ API ਐਂਡਪੁਆਇੰਟ ਲਈ ਦਰ-ਸੀਮਾਵਾਂ ਲਾਗੂ ਕੀਤੀਆਂ ਜਾਂਦੀਆਂ ਹਨ।
