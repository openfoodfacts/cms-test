ਵਰਤਮਾਨ ਵਿੱਚ, ਅਸੀਂ ਇਤਿਹਾਸਕ ਡੇਟਾ ਡੰਪ (JSONL, MongoDB, CSV) ਦੀ ਪੇਸ਼ਕਸ਼ ਨਹੀਂ ਕਰਦੇ ਹਾਂ।

ਹਾਲਾਂਕਿ, ਵਿਅਕਤੀਗਤ ਉਤਪਾਦਾਂ ਲਈ, API ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਜਾਂ ਉਤਪਾਦ ਪੰਨੇ 'ਤੇ ਸੰਸ਼ੋਧਨਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਉਤਪਾਦ ਡੇਟਾ ਦੇ ਪਿਛਲੇ ਸੰਸਕਰਣਾਂ ਤੱਕ ਪਹੁੰਚ ਕਰਨਾ ਸੰਭਵ ਹੈ।

ਹਰ ਵਾਰ ਜਦੋਂ ਕੋਈ ਉਤਪਾਦ ਅੱਪਡੇਟ ਕੀਤਾ ਜਾਂਦਾ ਹੈ, ਤਾਂ ਇੱਕ ਨਵਾਂ ਸੰਸ਼ੋਧਨ (1 ਤੋਂ ਸ਼ੁਰੂ ਹੋਣ ਵਾਲਾ ਵਧਦਾ ਅੰਕ) ਬਣਾਇਆ ਜਾਂਦਾ ਹੈ।

ਉਦਾਹਰਨ ਲਈ, ਇਸ ਉਤਪਾਦ ਦਾ ਪਹਿਲਾ ਸੰਸ਼ੋਧਨ (=ਪਹਿਲਾ ਉਤਪਾਦ ਸੰਸਕਰਣ) ਪ੍ਰਾਪਤ ਕਰਨ ਲਈ, ਵਰਤੋਂ

https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1.
