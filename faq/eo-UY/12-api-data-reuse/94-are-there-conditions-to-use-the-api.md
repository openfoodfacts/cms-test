La tutan dokumentaron pri API-uzado oni povas trovi sur la API-dokumentara paĝo, sed jen rapida resumo:

- La datumbazo Open Food Facts estas havebla kiel malfermaj datumoj laŭ la Open Database License (ODbL), vidu https://world.openfoodfacts.org/terms-of-use por la juraj detaloj. La du kondiĉoj estas atribuo kaj samdivido. Se vi kombinas datumojn el Open Food Facts kun aliaj datumbazoj, tiam la ODbL postulas, ke la rezulta datumbazo estu publikigita ankaŭ kiel malfermaj datumoj. Ĝi ankaŭ signifas, ke vi povas kombini la datumojn nur kun fontoj, kiuj permesus tian redistribuon.

- Vi ĉiam devas uzi kutiman uzanto-agenton dum plenumado de API-vokoj por identigi vian aplikaĵon.

- Rapido-limoj estas devigitaj por ĉiu API-finpunkto.
