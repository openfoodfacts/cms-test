దురదృష్టవశాత్తు ఉత్పత్తి పేరు మీద మాత్రమే సులభంగా మరియు ఖచ్చితంగా API ద్వారా శోధించడం ఇంకా సాధ్యం కాలేదు.

వర్గంపై ఫిల్టర్‌ని ఉపయోగించడం వల్ల మీ శోధనను మరింత ఖచ్చితమైనదిగా చేయవచ్చు.
