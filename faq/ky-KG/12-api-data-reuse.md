## Жүктөлгөн сүрөттөр үчүн кандай өлчөм жакшы болору боюнча документтердин кайсы бир жеринде сунуштар барбы?

Бул тармактын жай же кымбат болушу өлкөлөргө жараша болушу мүмкүн. Салмагы же бою 5000 пикселден жогору болгон нерсе анчалык деле пайдалуу эмес болушу мүмкүн. жана эгер сиз кандайдыр бир жол менен тармактын жай иштеп жатканын аныктай алсаңыз, анда 2000 пикселдик сүрөт да сонун болмок (албетте, сүрөттүн жоктугунан жакшыраак!)

## Штрих-кодсуз тамак-аш жөнүндө эмне айтууга болот?

Ачык азык-түлүк фактылары таңгакталган азык-түлүк жөнүндө гана маалыматты камтыйт. Жемиштердин (мисалы, помидор же банан) жана башка азык-түлүктөрдүн орточо баалуулуктары үчүн, сиз расмий улуттук тамактануу маалымат базаларынын бирин колдоно аласыз.

**Эскертүү:** Төмөндөгү тизмеде эң маанилүү улуттук азык-түлүк маалымат базаларынын айрымдары камтылган. Эгер сиз тизмеге башка маалымат базасын кошуу керек деп ойлосоңуз, биз менен төмөнкү дарек боюнча байланышыңыз: [https://world.openfoodfacts.org/contact](https://world.openfoodfacts.org/contact)

**Улуттук азык-түлүк маалымат базаларынын тизмеси**

## API аркылуу так продукт аталышын издей аламбы?

Тилекке каршы, API аркылуу продукттун аталышы боюнча гана жана так издөө азырынча мүмкүн эмес.

Бирок, категория боюнча чыпканы колдонуу издөөңүздү так кылууга жардам берет.

## Долбоорлорум үчүн маалыматтарды кантип алсам/чогулта алсам болот?

"Ачык тамак-аш фактыларынын" башкы барагында, экрандын жогорку сол бурчунда жылдыруучу меню бар. Анын эң төмөн жагында сиз чыкылдата турган "өркүндөтүлгөн издөө" опциясын таба аласыз. Андан кийин, кайсы критерийлер сиздин долбооруңузга (долбоорлоруңузга) эң тиешелүү экенин аныктоо сизден көз каранды. Тандалгандан кийин, барактын ылдый жагында ылдый жылдырып, "Жыйынтыктарды жүктөп алуу" баскычын чыкылдатуу менен алынган натыйжаларды жүктөп ала аласыз.

Ошондой эле төмөнкүлөргө кайрылсаңыз болот:

- Биздин API документтерибиз:[ https://openfoodfacts.github.io/api-documentation/](https://openfoodfacts.github.io/api-documentation/)
- Open Food Facts' колдонуу шарттары:[ ](https://world.openfoodfacts.org/terms-of-use)[https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use)
- Биздин маалыматтар боюнча:[ https://world.openfoodfacts.org/data](https://world.openfoodfacts.org/data)

## API колдонуу үчүн шарттар барбы?

API колдонуу боюнча бардык документтерди [API документтеринин барагынан](https://openfoodfacts.github.io/openfoodfacts-server/api/) тапса болот, бирок бул жерде кыскача маалымат берилген:

- Ачык азык-түлүк фактылары базасы Ачык маалымат базасынын лицензиясы (ODbL) боюнча ачык маалыматтар катары жеткиликтүү, юридикалык маалымат алуу үчүн [https://world.openfoodfacts.org/terms-of-use](https://world.openfoodfacts.org/terms-of-use) караңыз. Эки шарт - атрибуция жана бирдей бөлүшүү. Эгер сиз Open Food Facts маалымат базасынан алынган маалыматтарды башка маалымат базалары менен бириктирсеңиз, анда ODbL алынган маалымат базасын ачык маалыматтар катары да жарыялоону талап кылат. Ошондой эле, сиз маалыматтарды ушундай кайра бөлүштүрүүгө мүмкүндүк берген булактар менен гана бириктире аласыз дегенди билдирет.

- Колдонмоңузду аныктоо үчүн API чалууларын аткарып жатканда, сиз **ар дайым** колдонуучу агентин колдонушуңуз керек.

- Ар бир API акыркы чекити үчүн тарифтик чектөөлөр колдонулат.

## Тарыхый маалыматтарга кантип кире алам?

Учурда биз тарыхый маалыматтардын демпин (JSONL, MongoDB, CSV) сунуштабайбыз.

Бирок, айрым өнүмдөр үчүн, продукт маалыматтарынын мурунку версияларына API аркылуу же продукт барагында редакцияларды колдонуп кирүүгө болот.

Продукция ар бир жолу жаңыртылган сайын, жаңы редакция (1ден баштап көбөйүүчү сан) түзүлөт.

Мисалы, бул продуктунун биринчи редакциясын (=биринчи продукт версиясын) алуу үчүн, колдонуңуз

[https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1](https://world.openfoodfacts.org/product/7623186089763/joghurt-baumnuss-migros?rev=1).

Ошо сыяктуу эле, rev параметрин API менен колдонсо болот:

[https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1](https://world.openfoodfacts.org/api/v2/product/7623186089763?rev=1)
