Alle documentatie over het gebruik van de API is te vinden op de API-documentatiepagina, maar hier is een korte samenvatting:

- De Open Food Facts-database is beschikbaar als open data onder de Open Database License (ODbL). Zie https://world.openfoodfacts.org/terms-of-use voor de juridische details. De twee voorwaarden zijn toeschrijving en gelijk delen. Als je gegevens uit Open Food Facts combineert met andere databases, vereist de ODbL dat de resulterende database ook als open data wordt vrijgegeven. Dit betekent ook dat je de gegevens alleen kunt combineren met bronnen die een dergelijke herverdeling toestaan.

- Bij API-aanroepen moet u altijd een aangepaste User-Agent gebruiken om uw app te identificeren.

- Voor elk API-eindpunt gelden snelheidslimieten.
